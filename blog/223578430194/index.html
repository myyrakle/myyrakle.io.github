[Rust] кј¬лҰ¬ нҳём¶ң мөңм Ғнҷ” (Tail Call Optimization)

  • 2024.09.09
  • ::
  • 2 min read
#Rust
[мӣҗліё л§ҒнҒ¬]

кј¬лҰ¬ нҳём¶ң мөңм Ғнҷ”(Tail Call Optimization)лҠ” мһ¬к·Җ нҳём¶ң н‘ңнҳ„лІ•м—җм„ң мӮ¬мҡ©лҗҳлҠ” л©”лӘЁлҰ¬/м„ұлҠҘ л№„нҡЁмңЁм„ұмқ„ мҷ„нҷ”н•  мҲҳ мһҲлҠ” мөңм Ғнҷ” кё°лІ• мӨ‘ н•ҳлӮҳлӢӨ.

м—¬кё°м„ңлҠ” н”јліҙлӮҳм№ҳлҘј мҳҲлЎң л“Өм–ҙліҙкІ лӢӨ.

fn fibonacci_naive(n: u32) -> u32 {
    if n == 0 {
        0
    } else if n == 1 {
        1
    } else {
        fibonacci_naive(n - 1) + fibonacci_naive(n - 2)
    }
}

мқҙлҹ° мҲҳн•ҷм Ғмқҙкі  мһ¬к·Җм Ғмқё н‘ңнҳ„мқҙ м Ғн•©н•ң кІҪмҡ°м—җлҠ” "н‘ңнҳ„м„ұ"кіј "к°„кІ°н•Ё" мёЎл©ҙм—җм„ңлҠ” мһ¬к·Җнҳём¶ң н‘ңнҳ„мқҙ лӣ°м–ҙлӮҳлӢӨ.
к·јлҚ° к°„кІ°н•ҳлӢӨкі  н•ҙм„ң нҡЁмңЁм ҒмқҙлһҖкұҙ м•„лӢҲлӢӨ.
мһ¬к·Җн•ЁмҲҳлҠ” нҠ№м„ұмғҒ н•ЁмҲҳ нҳём¶ң кіјм •мқҙ көүмһҘнһҲ л§Һмқҙ мқјм–ҙлӮҳлҠ”лҚ°, н•ЁмҲҳ нҳём¶ңмқҖ н•„м—°м ҒмңјлЎң мҠӨнғқмқ„ м¶”к°Җм ҒмңјлЎң мҢ“кё° л•Ңл¬ём—җ мҠӨнғқ л©”лӘЁлҰ¬лҘј л№„нҡЁмңЁм ҒмңјлЎң лӮӯл№„н•ңлӢӨ. кІҢлӢӨк°Җ мҠӨнғқ мҢ“кі  лӮҙлҰ¬кі  н•ҳлҠ” кІғлҸ„ мӢӨн–үмҶҚлҸ„м—җ мҳҒн–Ҙмқ„ мӨҖлӢӨ.
к·ёлһҳм„ң мһ¬к·Җнҳём¶ңмқҖ лЈЁн”„мқҳ л°ҳліөліҙлӢӨлҠ” нӣЁм”¬ лҠҗлҰ¬лӢӨ.

м Җ мҳҲм ң мҪ”л“ңмқҳ кІҪмҡ°м—җлҸ„ н•ң 50 м •лҸ„к№Ңм§ҖлҠ” к·ёлҹӯм Җлҹӯ мӢӨн–үмқҙ лҗҳм§Җл§Ң, 60мқ„ л„ҳм–ҙк°Җл©ҙ м—„мІӯлӮҳкІҢ лҠҗл Өм§Ҳ кІғмқҙкі , к·ёліҙлӢӨ лҚ” л„ҳм–ҙк°Җл©ҙ мҠӨнғқмҳӨлІ„н”ҢлЎңлЎң н„°м§Җкё° л”ұ мўӢлӢӨ.

м Җлҹ° нҳ•нғңмқҳ мҪ”л“ңлҠ” м–ҙм…Ҳлё”лҰ¬лҘј к№ҢлҙҗлҸ„

к·ёлғҘ мҲңм§„н•ҳкІҢ мҠӨмҠӨлЎңлҘј нҳём¶ңн•ңлӢӨ.




кј¬лҰ¬ нҳём¶ң мөңм Ғнҷ”

кј¬лҰ¬ нҳём¶ң мөңм Ғнҷ”лҠ” нҠ№мҲҳн•ҳкІҢ м •мқҳлҗң нҳ•нғңмқҳ мһ¬к·Җнҳём¶ң н•ЁмҲҳлҘј лЈЁн”„лЎң м№ҳнҷҳн•ҳлҠ” мөңм Ғнҷ” кё°лІ•мқҙлӢӨ.
мһ¬к·Җнҳём¶ңкіј л°ҳліөл¬ёмқҳ мһҘлӢЁм җмқ„ м Ғм ҲнһҲ м Ҳм¶©н•ң нҳ•нғңлқјкі  н•  мҲҳ мһҲлӢӨ.

к·ёлҰ¬кі  RustлҠ” м•”мӢңм ҒмңјлЎң кј¬лҰ¬ нҳём¶ң мөңм Ғнҷ”лҘј м§Җмӣҗн•ңлӢӨ. м§Ғм ‘ кө¬нҳ„н•ңкұҙ м•„лӢҲкі  к·ёлғҘ LLVMм—җ м§¬л•ҢлҰ° кІғ к°ҷлӢӨ.

кј¬лҰ¬нҳём¶ңмқҖ лҢҖлһө лӢӨмқҢкіј к°ҷмқҖ нҳ•нғңлҘј к°Җм§„лӢӨ.

#[inline(never)]
fn fibonacci_tail(n: u128, a: u128, b: u128) -> u128 {
    if n == 0 {
        a
    } else {
        fibonacci_tail(n - 1, b, a + b)
    }
}

#[inline(never)]
fn fibonacci(n: u128) -> u128 {
    fibonacci_tail(n, 0, 1)
}

мІҳмқҢ ліҙл©ҙ мқҙн•ҙн•ҳкё° мүҪм§Җ м•Ҡмқ„ мҲҳлҸ„ мһҲлӢӨ.
мӣҗлһҳ лҰ¬н„ҙк°’мқ„ нҶөн•ҙ мғҒмң„ нҳём¶ңмһҗлЎң кІ°кіјлҘј л„ҳкё°лҚҳ кІғмқ„, мқҙм ңлҠ” нҢҢлқјлҜён„°лҘј нҶөн•ҙ кі„мҠ№н•ҳлҸ„лЎқ л°”лҖҢм—ҲлӢӨ.

к·ёлһҳм„ң мқҙм ң мқҙкұё м–ҙм…ҲмңјлЎң к№Ңліҙл©ҙ

мқҙм ңлҠ” мһ¬к·Җнҳём¶ңмқҙ м „л¶Җ лқјмӮ¬м§Җкі  лӢЁмҲңн•ң м җн”„ кө¬л¬ёл§Ң лӮЁм•ҳлӢӨ.
лЈЁн”„лЎң м№ҳнҷҳлҗң кІғмқҙлӢӨ.

мқҙлҹ¬л©ҙ мқҙм ң Stackoverflow л¬ём ңлҸ„ м—Ҷм–ҙмЎҢмқ„лҝҗл§Ң м•„лӢҲлқј, м„ұлҠҘлҸ„ м••лҸ„м ҒмңјлЎң мҰқк°Җн•ңлӢӨ.

мӣҗлһҳлҠ” 60л§Ң л„ҳкІЁлҸ„ кұ°мқҳ л»—лҚҳ кІғмқҙ, 1000мқ„ л„ҳкІҢ л„ҳкІЁлҸ„ 1мҙҲ лҜёл§ҢмңјлЎң мҷ„лЈҢлҗңлӢӨ.




ліҙмһҘн•ҳм§Җ м•ҠмқҢ (No Guarantee)

м•„м§Ғ RustлҠ” кј¬лҰ¬ нҳём¶ң мөңм Ғнҷ”лҘј лӘ…мӢңм ҒмңјлЎң м§Җмӣҗн•ҳм§ҖлҠ” м•ҠлҠ”лӢӨ.
LLVM мөңм Ғнҷ” нҢЁмҠӨм—җ кё°лҢҖм„ң м–ҙм©ҢлӢӨліҙлӢҲ лӮҳмҳӨкІҢлҒ” лҗҳм–ҙмһҲлҠ” кұ°м§Җ, Rustк°Җ лӘ…мӢңм Ғмқё мөңм Ғнҷ” ліҙмһҘмқ„ м ңкіөн•ҙмЈјкұ°лӮҳ н•ҳлҠ”кұҙ м—ҶлҠ” кІғмқҙлӢӨ.

мҳҲм „м—җ Tail Callм—җ лҢҖн•ң ліҙмһҘмқ„ л¬ёлІ•м ҒмңјлЎң м§Җмӣҗн•ҳл Өкі  be/become к°ҷмқҖ нӮӨмӣҢл“ңлҘј мҳҲм•Ҫн•ҙл‘җкёҙ н–ҲлҠ”лҚ°, нҳ„мһ¬лЎңм„ңлҠ” мҡ”мӣҗн•ҳлӢӨ. мҡ°м„ мҲңмң„к°Җ көүмһҘнһҲ лӮ®кІҢ мһЎнҳҖм„ң мһ‘м—…мқҙ м „нҳҖ м§„н–үлҗҳм§Җ м•Ҡкі  мһҲлҠ” кІғ к°ҷлӢӨ.
https://rust-lang.github.io/rfcs/0601-replace-be-with-become.html

м•„л¬ҙнҠј к·ёлһҳм„ң Tail Call Optimizationмқҙ м Ғмҡ©лҗҳм—ҲлҠ”м§ҖлҘј кІҖмҰқн•ҳлҠ” к°ҖмһҘ нҷ•мӢӨн•ң л°©лІ•мқҖ, мң„мІҳлҹј м–ҙм…Ҳлё”лҰ¬лҘј к№ҢліҙлҠ” кІғмқҙлӢӨ.

нҳ№мқҖ л§ӨнҒ¬лЎң кё°л°ҳмңјлЎң Tail Callмқҙ мһҳ м Ғмҡ©лҗҳлҠ” нҳ•нғңмқҳ м •нҳ•нҷ”лҗҳ мҪ”л“ңлҘј мғқм„ұн•ҙмЈјлҠ” crateмқҙ н•ҳлӮҳ мһҲлӢӨ.

cargo watch는 live reload 기능을 제공하는 보조도구다.
쉽게 말해서, 소스코드를 수정하고 저장할때마다 프로그램을 재실행시켜주는 용도라고 보면 된다.




설치

cargo를 통해서 간단하게 설치할 수 있다.

cargo install cargo-watch

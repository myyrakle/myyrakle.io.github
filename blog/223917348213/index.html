[Linux] Arch: AUR Helper - yay

  • 2025.07.01
  • ::
  • 1 min read
#Linux
[원본 링크]

pacman이 Arch에서 직접 관리하고 제공하는 오피셜 패키지 시스템이라면, AUR(Arch User Repository)는 그 외 유저들이 자발적으로 관리하는 사설 패키지라고 볼 수 있다.
비교적 마이너하고 사소한 패키지들은 pacman에 없고 AUR에 있다.

여기서는 yay라는 도구를 통해서 AUR 패키지를 하나 설치해보겠다.
yay는 보통 직접 컴파일해서 설치하는 것이 권장된다.

sudo pacman -S --needed base-devel git
git clone https://aur.archlinux.org/yay.git
cd yay
makepkg -si

적당히 따라 쳐서


이렇게 실행이 되면 잘 설치된 것이다.


그럼 필요한 패키지를 찾아서 직접 설치를 해볼 수 있다.
CLI 인터페이스는 pacman과 유사하다.


잘 깔렸다.

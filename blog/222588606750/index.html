[Influx DB] 데이터베이스

  • 2021.12.07
  • ::
  • 1 min read
#InfluxDB
[원본 링크]

인플럭스도 여타 RDB들처럼 기본적으로 데이터베이스를 가장 바깥 레이어로 깔고 시작한다.




데이터베이스 생성

데이터베이스의 생성은 create database 명령으로 수행할 수 있다.

create database 데이터베이스명

그럼 다음과 같이 데이터베이스가 생성될 것이다.
데이터베이스 목록은 show databases로 조회할 수 있다.




데이터베이스 접속

생성한 데이터베이스에 접속하려면 use 명령을 사용한다.

use 데이터베이스명




데이터베이스 삭제

데이터베이스를 삭제하는 것도 간단하다.
drop database 명령을 사용하면 된다.

drop database 데이터베이스명

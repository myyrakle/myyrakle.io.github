#Git
[원본 링크]

풀 리퀘스트는 협업 방식 중 하나다.
원본 프로젝트를 포크해서 작성한 후, 풀 리퀘스트를 날리면 관리자가 그걸 잘 반영하는 식으로 이루어진다.



포크

원본 레포지토리에 가서 저 포크버튼을 누르면 저게 복제돼서 내 레포지토리에 복제본이 생긴다.

그럼 이게 포크다.

이건 내 마음대로 사용할 수 있는 복제본이다.

저걸 클론떠다가 작업하면 된다.

클론뜨기 귀찮고 간단한 작업만 할거라면 그냥 github 편집기로 뚝딱 고쳐도 된다.

대충 작업하고

커밋푸시한다.

저기 위에 풀리퀘스트 버튼을 누른다.

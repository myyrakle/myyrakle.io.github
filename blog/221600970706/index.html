일반 분기 if-else문
if-else문은 자바와 사용법이 거의 동일하다. 뭐 똑같이 이렇게 쓸수있다.

다만 여기서 더 추가된 기능은 있다. if나 else 내의 구문을 반환식으로 사용할수도 있단것이다.
위의 코드를 반환식으로 변환하면 이렇게 된다.


패턴매칭 match
match는 자바의 switch문을 확장한 형태라고도 볼 수 있다. 더 자유롭고 기능이 풍부하기 때문이다.

대충 아래의 구조를 가진다. 값을 뒤에 안쓰고 앞에쓰는게 좀 특이하다.

매칭할_값 match
{
**    case 패턴1 => 문장1
**    case 패턴2 => 문장2
**    ...**
**    case 패턴N => 문장N**
}

간단한 예시다.
_는 와일드카드 패턴이라, 뭐든지 들어갈 수 있다는 뜻이다. default 레이블에 해당한다.

마찬가지로 반환식으로도 사용가능하다.

타입으로도 분기를 설정할 수 있다.
다만 패턴에 들어가는 타입은 집어넣은 변수의 타입이거나, 캐스팅가능한 서브타입이여야 한다.

패턴을 x:T 와 같이 쓰면, T 타입이거든 x에 바인딩한단 뜻이다. 변수 바인딩이 필요없다면 _를 써도 된다.

while문
이건 자바랑 똑같다.


for문
이건 자바랑 좀 다르다.
스칼라의 for문은 무조건 범위기반 반복문이다.
컬렉션을 순회하면서 왼쪽변수에 바인딩, 몸체부분을 수행한다.

스칼라의 for문은 아래의 문법구조를 가진다.

for(변수명 <- 컬렉션)
{ ... }

아래 코드는 그냥 다섯번 반복하면서 출력하는 간단한 코드다.

A until B는 A부터 B 전까지의 정수값을 가지는 범위표현식이다.

B의 값까지 포함시키고싶다면 until 대신 to를 쓰면 된다.



다중 for문
스깔라에서는 놀랍게도 하나의 for문에 2개 이상의 for문을 끼워넣을수 있다.
방법은 간단하다. 기존의 for 구문을 ;으로 구분해서 끼워넣으면 된다.
그럼 맨 앞에있는 평가식부터 가장 바깥의 반복문으로 처리가 된다.

for( i <- 컬렉션1; j <- 컬렉션2 ...)

3개 이상도 중첩할수 있다.
쓸일이 많을것같진 않다만...

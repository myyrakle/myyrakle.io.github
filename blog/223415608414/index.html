제한사항

Rust의 프로파일링은 주로 pprof-rs를 통해 이루어진다.
거의 Linux 환경에서만 제대로 사용이 가능하다.

메모리 사용량 모니터링은 아직 지원되는게 없는 모양이다.




설치

필요한 종속성은 2가지다.
설치해준다.

cargo add pyroscope
cargo add pyroscope_pprofrs

이정도다.




코드 구성

아래와 같은 axum 서버 코드에 pyroscope 클라이언트를 구성해보겠다.

코드 형태는 대략 다음과 같다.

fn setup_pyroscope() {
    use pyroscope::PyroscopeAgent;
    use pyroscope_pprofrs::{pprof_backend, PprofConfig};

    // Configure profiling backend
    let pprof_config = PprofConfig::new().sample_rate(100);
    let backend_impl = pprof_backend(pprof_config);

    // Configure Pyroscope Agent
    let agent = PyroscopeAgent::builder("http://localhost:4040", "myapp")
        .backend(backend_impl)
        .build()
        .unwrap();

    agent.start().unwrap();
}

pprof 샘플 주기를 지정하고, pyroscope 서버와 앱 이름을 정해서 넣어주면 된다.
그리고 start로 실행한다. 이 함수는 nonblock이라서 가장 먼저 실행하면 된다.

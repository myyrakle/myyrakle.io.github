Ubuntu에서 엔비디아 드라이버를 구성하는 방법을 간단히 정리한다.

먼저 nvidia 장치가 설치가 되어있는지부터 확인한다.

lspci | grep -i nvidia

이런 식으로 뜨면 잘 구성이 되어있는 것이다.

nvidia-smi 명령이 설치되어있는지 확인하고

nvidia-smi

없다면 설치한다.

sudo apt-get install ubuntu-drivers-common -y

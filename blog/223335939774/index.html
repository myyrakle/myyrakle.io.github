CDK는 Cloud Development Kit의 축약으로, AWS에서 직접 제공하는 IaC 도구다.
코드로 인프라를 관리한다는 것이다. 테라폼 같은 도구들과 비교할 수 있겠다. 내부적으로는 AWS Cloudformation을 사용해서 상태 관리를 한다.
https://blog.naver.com/sssang97/222801645516

언어는 Node.js, Python, C#, Go, Java 정도가 제공되나, 파이썬을 가장 많이 쓰고 공식 레포에 예제가 많은 것도 파이썬인 것 같다. 그래서 나도 파이썬으로 하겠다.




cdk 기본구성

사전에 aws-cli 설치와 aws configure가 전부 되어있어야 한다.

CLI는 npm으로 설치한다.

npm install -g aws-cdk

이렇게 깔고

부트스트랩 명령을 날려서 기본 cloudformation 세팅을 해준다.

그럼 이런식으로 중앙 구성이 생성된다.
이건 일종의 컨트롤 플레인이다. 신경쓰지 않아도 된다.

코드에 스택이 여러개일때는 --all 옵션을 줘서 다 실행시키거나 식별자를 직접 지정해야 하지만, 단일 스택일때는 아무 파라미터도 주지 않아도 된다.

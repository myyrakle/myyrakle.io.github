[Java] 열거

  • 2019.11.29
  • ::
  • 1 min read
#Java
[원본 링크]

열거는 일련의 상수값을 포함하는 타입을 정의하는 것이다.
enum 열거체명
{
**    열거명1, 열거명2, ...**
}

보통 switch 문 등과 함께 곧잘 사용한다.

열거는 이름으로만 식별되는 단순한 상수지만, 사실 내부적으로는 정수로 처리한다.
내부의 값은 열거값의 ordinal 메소드로 땡겨올 수 있다.

0부터 시작하는걸 알 수 있다.

그렇다.

  • ::

[원본 링크]

php도 버전 5부터 클래스 문법이 지원된다.

문법구조는 대략 c++ 등의 oop 언어들과 흡사하다.

멤버변수 선언은 그냥 변수선언과 동일하다. 그놈의 달러표시만 열심히 붙이면 된다.

멤버함수도 그냥 함수선언과 동일하다.

그리고 생성자/소멸자는 __construct/__destruct 키워드로 선언된다. 생성자나 기타 메서드들도 오버로딩이 불가능하다.

객체 내에서 멤버변수 등의 접근은 $this->변수명; 으로 표현되는데, 뒤에는 $를 붙이면 안된다. 참 일관성도 없다.

접근제어자도 여타 언어들과 다를바 없다.
 private/protected/public이며, 붙이지 않았을 경우의 디폴트제어는 public이다.

클래스의 객체를 생성하려면 **new 클래스명(생성자인자)**를 하면 되고,
멤버에 접근하려면 객체->멤버명으로 쓰면 된다.

#Python
[원본 링크]

파이썬에선 분기문을 위한 키워드로 if, else, elif를 제공한다.

if문은 아래의 구조를 가진다.
if** 조건식:**
**    구문...**
들여쓰기를 잊지 말자...

if 내의 구문은 조건식이 참일 때만 작동한다.

이렇게.

거짓일 경우도 핸들링하고 싶다면 else를 쓰면 된다. 마찬가지로 : 붙이고 들여쓰기로 구분한다.

if-else를 중첩해서 여러가지의 조건을 핸들링하고 싶다면, elif를 쓸 수 있다.
if와 마찬가지로 조건식을 넣고 :, 구문을 넣는다.

잘 돈다.


삼항 연산자?
파이썬도 삼항연산자와 유사한 표현식을 제공한다.
if-else문을 표현식의 형태로 가공한 것인데... 모양이 좀 많이 못생겼다.

참값 if 조건식 else 거짓값

조건식이 참일경우 앞의 참값을 반환하고, 거짓이면 뒤의 거짓값을 반환한다.

잘 된다.


스위치문이나 패턴매칭은 없다.
불편하다.

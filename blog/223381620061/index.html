[PostgreSQL] PostGIS: 설치

  • 2024.03.13
  • ::
  • 1 min read
#PostgreSQL
[원본 링크]

PostGIS는 PostgreSQL의 확장 중 하나로, 복잡한 좌표기반의 지리적 계산 기능을 제공하는 지리 정보 시스템(GIS:Geographic information system)이다.

오픈소스 진영에서는 이걸 따라오는 GIS가 딱히 없는 것으로 알고 있다.




설치하기 (Windows)

이 기능은 PostgreSQL 기본 스펙에는 포함되어있지 않다. 확장 프로그램 형태로 설치해서 사용해야 한다.
Windows와 unix, linux 계열 모두 지원이 된다.

Windows의 경우에는, installer로 설치했다면 stackbuilder라는 패키지 관리자가 있을 것이다.
그걸 실행해서

PostGIS를 고르고

설치하면 된다.




설치하기 (Ubuntu)

아래 명령으로 저장소에 postgis를 추가한다.

sudo sh -c 'echo "deb http://apt.postgresql.org/pub/repos/apt xenial-pgdg main" >> /etc/apt/sources.list'
wget --quiet -O - http://apt.postgresql.org/pub/repos/apt/ACCC4CF8.asc | sudo apt-key add -
sudo apt update

그리고 버전에 맞는 postgis를 설치한다.
내 경우에는 postgres 버전이 14였다.

sudo apt install postgresql-14-postgis




확장 설치하기

postgis 파일을 설치받았다면, 해당 정보들을 관리하기 위한 데이터베이스와 스키마를 추가하고 여기에 create extention으로 postgis 확장을 셋업해준다.

CREATE DATABASE gisdb;
\connect gisdb;

CREATE SCHEMA postgis;
ALTER DATABASE gisdb SET search_path=public, postgis, contrib;
\connect gisdb;  
CREATE EXTENSION postgis SCHEMA postgis;
SELECT postgis_full_version();

그래서 오류 없이 처리되면 잘 된 것이다.

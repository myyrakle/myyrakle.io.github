자바에는 기본적으로 json이 제공되지 않는다.
때문에 외부 라이브러리를 땡겨써야 하는데, 개중에서 제일 나은게 구글에서 만든 Gson이라고 한다.

jar 다운로드
https://mvnrepository.com/artifact/com.google.code.gson/gson/2.8.5

레퍼런스
https://www.javadoc.io/doc/com.google.code.gson/gson/2.8.5

여튼, 이걸 땡겨써도 Json을 쓰는게 꽤 까다로웠다.

아래는 python 서버에서 보내는 json을 받아 처리하는 클라이언트 소스다.
앞서 포스팅했던 Python Json 글에서 이어진다.

import java.net.*;
import java.io.*;
import com.google.gson.*;
import com.google.gson.stream.*;

//변환을 위한 실제 타입. 
//Js, Python과 달리 동적 타입이 지원되지 않기 때문에 필요함.
class Person
{
    public String name;
    public String age;
    public String[] family;
}

public class Main
{
    final static String server_address = "서버 아이피";
    final static int port = 12345;

    public static void main(String[] args)
    {
        try
        {
            //연결 후 읽어옴
            var socket = new Socket(server_address, port);
            var socket_reader = socket.getInputStream();
            var buffer = new byte[1000];
            socket_reader.read(buffer, 0, 1000);
            var json_string = new String(buffer, "UTF-8");

            System.out.println(json_string); //확인

            var json_reader = new JsonReader(new StringReader(json_string)); //String으로 바로 보내는것도 있는데 그거 쓰면 에러남. 덮어줘야 함.
            Person result = new Gson().fromJson(json_reader, Person.class); //Person 타입으로 실체화. 변수에 타입 명시 필요

            System.out.println(result.family[1]); //확인

            socket.close();
        }
        catch(Exception e)
        {
            e.printStackTrace();
        }
    }
}

여기서는 Person이라는 명확한 타입으로 실체화를 했는데, Json의 동적 타입에 맞게 추상화된 타입인 JsonObject로 변환해서 쓸 수도 있다.
new JsonParser().parse(json문자열).getAsJsonObject()...
대신 이놈은 명확한 타입을 알아내지 못하기 때문에 요소를 get할때마다 타입 명시를 해줘야한다.

이 JsonObject라는 놈을 한번 처음부터 생성을 해보자

import com.google.gson.*;
import com.google.gson.stream.*;

public class Main
{
    public static void main(String[] args)
    {
        try
        {
            var json = new JsonObject();

            json.addProperty("name", "경직");
            json.addProperty("age", 65536);
            json.addProperty("nation","korea");

            System.out.println(json.toString());
        }
        catch(Exception e)
        {
            e.printStackTrace();
        }
    }
}


JsonObject에 다른 JsonObject를 끼워넣을 수도 있다. 이것만 메서드 이름이 다르다. add.

import com.google.gson.*;
import com.google.gson.stream.*;

public class Main
{
    public static void main(String[] args)
    {
        try
        {
            var json = new JsonObject();

            json.addProperty("name", "경직");
            json.addProperty("age", 65536);
            json.addProperty("nation","korea");

            var child = new JsonObject();
            child.addProperty("으아아", "않되");
            child.addProperty("내 뒤를", "바리스타");

            //이어붙이기O
            json.add("child",child);

            System.out.println(json.toString());
        }
        catch(Exception e)
        {
            e.printStackTrace();
        }
    }
}


요소를 가져오려면 앞서 언급했듯이 계속 타입 명시를 해줘야 한다.

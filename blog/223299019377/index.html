gorm은 go의 대표적인 ORM 라이브러리 중 하나다.
호불호가 많이 갈리긴 하는데, 그럼에도 대표적으로 많이 쓰인다.



설치

기본 모듈을 설치하고

go get -u gorm.io/gorm

여느 DB 모듈들이 그렇듯, 사용할 DB의 드라이버도 깔아야 한다.

go get -u gorm.io/driver/postgres

난 postgres다.




데이터베이스 연결

db 연결은 다음과 같은 형태로 할 수 있다.

    databaseURL := "host=localhost user=postgres password=... dbname=postgres port=5432 sslmode=disable"

	db, err := gorm.Open(postgres.Open(databaseURL), &gorm.Config{})
	if err != nil {
		panic(err)
	}

저 db 객체 가져다가 이것저것 돌려쓰면 된다.

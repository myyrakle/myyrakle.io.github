Rust 1.65부터 추가된 기능이다.

Result와 Option 등을 이용해 예외처리를 하다보면, 아래와 같은 패턴을 사용할 일이 잦다.

Option을 까보다가 있으면 깐 값을 그대로 받아서 사용하고, 없다면 오류값 등을 Option::None이나 Result::Err를 바로 반환해서 함수를 탈출하는 것이다.

그냥 보면 잘 작동하고, 괜찮아보일 수도 있지만, 이걸 반복하다보면 좀 불편하다.
어차피 내부 값을 그대로 받아올 것인데도 분해식으로 x 변수 정의하고, 그걸 리턴해서 받아오고...

그래서 이런 것에 대한 함축적인 sugar 신택스로 나온게 let-else문이다.
위의 코드를 let-else로 바꾸면 이런 코드가 된다.

훨씬 간결해진 것을 볼 수 있다.
else문에는 제어권에 대한 코드만 적절히 써주면 된다.

#GCP
[원본 링크]

Cloud Storage는 GCP에서 제공하는 파일 저장용 서비스다.
AWS의 S3에 대응된다고 보면 된다.




비용

전반적으로 AWS보다 약간 싸거나 비슷하다.
https://cloud.google.com/storage/pricing?hl=ko

예를 들어, 서울 리전에 스토리지를 두면 GCP는 GB당 0.23 달러, AWS는 0.25 달러가 나온다.

쓰기나 읽기 작업에 대해서는 AWS와 다른 방식으로 요금이 부과된다. AWS는 전송 데이터 크기로 매기는데, 이건 작업 횟수로 매긴다.




버킷 생성

스토리지를 이용하려면 먼저 버킷 단위를 만들어야 한다.

버킷 이름은 전 세계에 고유한 이름으로 지어야 한다.
이게 서브도메인중 하나가 될 것이기 때문이다.

리전을 선택한다.

매우 높은 고가용성을 원한다면 멀티리전을 사용하는 것이 좋다. 물론 약간 더 비싸다.


데이터 스토리지는 특별한 일이 없다면 스탠다드가 무난하다.

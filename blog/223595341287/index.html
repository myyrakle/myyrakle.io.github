musica 리메이크 v0.1.0 배포

  • 2024.09.25
  • ::
  • 1 min read
#개발노트
[원본 링크]

https://github.com/myyrakle/musica

내가 최근에 데탑 새로 맞추고 우분투 24를 깔았는데, 제대로 돌아가는 뮤직 플레이어가 없어서 자급자족하려고 만들었다. 리듬박스인가 그거도 24 이전까지만 지원되었던거같다.

예전에 이런 용도로 fltk-rs를 써서 대충 만들었던게 있었는데
https://github.com/myyrakle/musica_legacy
코드도 거지같이 나오고, 플랫폼 호환성도 생각보다 별로였던 것 같아서 iced 써서 다시 만들었다.




기본 사용법

설치는 cargo로 받아도 되고,

cargo install musica-app
musica-app

릴리즈 페이지에서 받아도 된다.
https://github.com/myyrakle/musica/releases/tag/v0.1.0
windows는 빌드를 안해봐서 모르겠는데.. 아마 되지 않을까?
mac은 디렉터리를 다이얼로그 열어서 선택하는 것만 빼면 잘 돌 것이다. 이건 갖다쓴 native dialog 라이브러리에 버그가 있어서 고쳐야한다.

UI는 최대한 단순하게 만들었다.

이전곡, 다음곡, 일시정지/재시작 정도만 있고


세팅 모달로 정말 기본적인 것만 제어할 수 있다.
랜덤모드로 켜면 랜덤으로 재생한다.

그리고 저 경로에 있는 것만 로드해서 재생한다.

추가할만한게 많긴 한데, 나는 이정도만 해도 충분해서 언제 기능을 또 추가할지는 모르겠다.

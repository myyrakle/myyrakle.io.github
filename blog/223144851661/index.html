[Rust] Slint: 시작하기

  • 2023.07.02
  • ::
  • 2 min read
#Rust
[원본 링크]

Slint는 Rust로 구현된 GUI 개발 툴킷 라이브러리다.
최근에 1.0 버전이 배포되었다.

윈도우즈, Mac, Linux 멀티플랫폼을 지원하며 웹어셈블리 기반의 웹페이지도 지원한다.
뭔지는 모르겠지만 임베디드도 지원한다고 한다.

개발 언어는 Rust와 C++을 지원하고, Javascript는 실험 단계인 것 같다. 가장 잘 지원되는건 Rust다.

구조는 뭐 이런식이라고 한다.

slint라는 자체 언어를 사용해서 UI 구조 등을 정의하고 실행한다는게 특징이다.
그래서 Rust를 쓰더라도 내부에 slint 코드를 정의해서 사용하는 형태로 응용이 된다.




라이센스

완전한 자유 오픈소스는 아니다.

공짜로 쓰고싶다면 GPLv3 라이센스에 기반해서 오픈소스를 개발하는데 써야하고,

상업용으로 사용한다면 유료 라이센스를 사용하면 된다.
월별 5유로부터 시작한다.
https://slint.dev/pricing

비용을 내고 싶지 않고, 오픈소스로 만들고 싶지도 않다면 로열티-free 라이센스라는 것을 적용할 수도 있다.
몇가지 귀찮은 요구사항을 충족시켜주면 된다.
프로그램 배포 사이트에 slint 써서 만든거라고 아이콘 달아주고 뭐 그런걸 하면 된다.

아래 링크에 들어가면 다 적혀있다.
https://github.com/slint-ui/slint/blob/master/FAQ.md#licensing




설치

기본 설정은 간단하다. 종속성 하나만 추가해주면 된다.

cargo add slint@1.1.0




Hello World 작성하기

간단하게 텍스트 하나 들어있는 윈도우 프로그램을 구현해보겠다.
slint는 매크로를 기반으로 윈도우 컴포넌트를 정의한다.

아래는 간단한 예시 코드다.

fn main() {
    let window = MainWindow::new().unwrap();

    window.run().unwrap()
}

slint::slint! {
    export component MainWindow inherits Window {
        height: 200px;
        width: 300px;

        Text {
            text: "hello world";
            color: green;
        }
    }
}

#Rust
[원본 링크]

sqlx는 Rust의 대표적인 DB용 라이브러리 중 하나다.
ORM은 아니고, 그냥 그.. 마이바티스 같은 쿼리 매퍼? 실행기다.

ORM 같은 거창한 기능은 없다.
그냥 쿼리에 구멍 뚫고 주입해서 쓰고, 반환값 좀 편하게 바인딩하고, async 기능이 잘 지원되는 것 정도다.
간단하게 쓰기는 좋다.




종속성

종속성은 다음과 같은 형태로 구성할 수 있다.
tokio를 쓰고, postgresql를 쓸거라면 이렇게 구성하면 된다.

[dependencies]
tokio = { version = "1.21.0", features = ["full", "time"] }
sqlx = { version = "0.7", features = [ "runtime-tokio", "postgres" ] }

이건 한번만 만들고 애플리케이션 전체에서 공유하면서 사용해야 한다.

그리고 bind로 구멍에 값을 넣은 다음 fetch를 사용해서 실행, 값을 가져오면 된다.

실제 DB 정보를 가지고서 쿼리를 검증하고, 타입과 일치하는지도 검증을 한다.

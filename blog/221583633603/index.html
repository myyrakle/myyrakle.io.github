if-else문
C와 거의 동일한데, 소괄호가 없다는 점이 다르다.

중괄호를 꼭 앞에다 붙여야하고, else도 바로 전의 if 구문이 끝나는 곳에다 이어붙여야한다.
거지같다.

그리고 선택적 구문이라는 if 확장구문이 있다.
if****구문; 논리식 {
    ...
}
와 같이 쓰면 저 구문을 수행하고 논리식을 수행하게 된다.
이 구문에서 변수를 선언할 경우, 그 변수는 if 스코프 내에서만 유효하다.

이건 if뿐만 아니라 switch문 등에서도 사용가능하다.


switch-case문
말은 스위치지만 패턴매칭구문이다.
모든 표현식을 case로 걸수 있고 break 없이도 자동탈출하며, 타입으로 분기를 걸수도 있다.


잘 돈다.

만약 전통적인 C 스위치처럼 탈출하지 않고 슥슥 내려가게 하려면 fallthrough를 붙이면 된다.

굳이 쓸일은 없을것같다만.

case에 콤마를 넣어서 여러값을 대응시킬 수도 있다.

switch에 인자부분을 제거하면 case에 조건식을 넣을 수도 있다. if-else와 사실상 동일하다.


타입 스위치다.

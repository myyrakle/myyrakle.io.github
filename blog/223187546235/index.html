역직렬화(Deserialize)

serde에서는 기본적으로, 아래와 같은 형태로 구조체 스키마를 정의해서 역직렬화를 한다.

별다른 문제는 없다.

근데 json 필드의 컨벤션이 rust의 식별자 컨벤션과 사뭇 다른 경우가 있을 수 있다.

이럴때는 rename 속성으로 명시를 해주는게 깔끔한 방법이다.




다형적인 타입에 대한 역직렬화

rust는 enum이라는 훌륭한 union 타입을 제공하기 때문에, 다형적인 타입에 대해서도 대응을 꽤나 유연하게 할 수 있다.
그래서 다음과 같은 형태로, 특정 json 필드의 타입이 이랬다 저랬다 바꾸는 것도 어렵지 않게 역직렬화를 할 수 있다.

방법은 그냥 enum으로 각 variant들을 정의해서 쑤셔넣는 것이다.

이러면 배열일 때와 단독 오브젝트일 때가 전부 커버된다.


잘 돌아간다.

예제 코드다.

fn main() {
    #[derive(serde::Deserialize, Debug)]
    struct TestItem {
        item_id: u32,
    }

    #[derive(serde::Deserialize, Debug)]
    #[serde(untagged)]
    enum TestResponseVariant {
        Object(TestItem),
        Array(Vec<TestItem>),
    }

    #[derive(serde::Deserialize, Debug)]
    struct TestResponse {
        id: u32,
        variant: TestResponseVariant,
    }

    {
        let json_raw = r#"{"id":12, "variant": { "item_id": 1234 }}"#;
        let response: TestResponse = serde_json::from_str(json_raw).unwrap();

        println!("{:?}", response);
    }

    {
        let json_raw = r#"{"id":12, "variant": [{ "item_id": 1234 }]}"#;
        let response: TestResponse = serde_json::from_str(json_raw).unwrap();

        println!("{:?}", response);
    }
}

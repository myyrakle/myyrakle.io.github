[k8s] kubectl 설치하기

  • 2023.04.24
  • ::
  • 1 min read
#Kubernetes
[원본 링크]

클러스터는 linux에만 설치가 가능하지만, 원격접속을 한다면 외부 windows나 linux에서도 접속을 해야할 일이 많을 것이다.




Windows에서 설치하는 법

choco 같은 매니저가 없다면 curl로 설치하는게 편하다.
설치하면

curl -o kubelet.exe "https://dl.k8s.io/release/v1.27.0/bin/windows/amd64/kubectl.exe"

바로 바이너리가 뜬다.

적절히 재배치해서 환경변수에 추가해주면 된다.




Unix에서 설치하는 법

다운받고

curl -LO "https://dl.k8s.io/release/$(curl -L -s https://dl.k8s.io/release/stable.txt)/bin/linux/amd64/kubectl"

install로 깔면 된다.

sudo install -o root -g root -m 0755 kubectl /usr/local/bin/kubectl



참조
https://kubernetes.io/ko/docs/tasks/tools/install-kubectl-linux/

linux 로컬 환경에서 meilisearch를 설치하고 간단하게 다뤄보는 방법을 정리한다.




설치

설치는 간단한 쉘코드 한줄로 끝낼 수 있다.

curl -L https://install.meilisearch.com | sh

그러면 이렇게, 실행한 시점의 경로를 기준으로 바이너리가 하나 생성될 것이다.

그래서 잘 실행되면 된 것이다.




서버 실행

이제 검색엔진 서버를 띄워보자.
아래와 같이 마스터키를 주소 실행시키면 정말 간단하게 뜬다. 패스워드는 16바이트 이상의 문자열이면 된다.

./meilisearch --master-key="패스워드..."

그러면 open된 호스트를 통해 웹 콘솔을 확인할 수도 있고,

그냥 rest API를 쏴서 상호작용을 할 수도 있다.




데이터 기본구조

meilisearch도 elastic search와 비슷한 스키마 구조를 따른다.
먼저 인덱스가 존재한다. 인덱스는 일종의 테이블 단위라고 할 수 있다.

그리고 인덱스에는 여러개의 document가 들어갈 수 있다. document 하나하나가 각각의 데이터 행이라고 보면 된다.
자세한건 별도 포스트에서 상세히 다루겠다.




샘플 데이터 넣어보기

meilisearch에서 공식적으로 제공하는 샘플을 넣어서 한번 어떻게 돌아가는지 보도록 하자.
언어는 nodejs를 사용하겠다. json 데이터 긁어오고 조회하기가 편해서 골랐다.

다운로드 => https://www.meilisearch.com/movies.json

이렇게 생겨먹었다.
복잡할 것은 없다. 고유번호로 id가 있고, 이미지 링크와 일반 텍스트, 정수 필드가 섞여있다.

그리고 데이터를 밀어넣어줄 코드를 작성하자.
우선 다음 모듈을 설치한다.

npm install meilisearch

코드는 다음과 같은 형태로 작성하면 된다.

const { MeiliSearch } = require("meilisearch");
const movies = require("./movies.json");

async function main() {
  const client = new MeiliSearch({
    host: "http://localhost:7700",
    apiKey: "q1w2e3r4asdf",
  });

  await client.index("movies").addDocuments(movies);
  console.log("done");
}

main();

"movies"라는 인덱스에 샘플 json 데이터를 그대로 밀어넣겠다는 선언이다.
인덱스는 자동으로 생성되고, 인덱스 스키마를 직접 정의할 필요도 없다. json 데이터를 읽어서 자동생성한다.

저대로 실행하고

약간 기다리면

이렇게 웹 콘솔에서 확인이 가능해질 것이다.

바로 간단하게 검색도 가능하다.

나는 이미지도 인식해서 자동으로 썸네일 띄워주는게 굉장히 신선했다.

코드에서도 간단하게 search를 수행할 수 있다.
아래 코드는 웹 콘솔에서 방금 했던 것과 완전하게 동일하게 검색을 수행하는 코드다.

const { MeiliSearch } = require("meilisearch");
const movies = require("./movies.json");

async function main() {
  const client = new MeiliSearch({
    host: "http://localhost:7700",
    apiKey: "q1w2e3r4asdf",
  });

  const result = await client.index("movies").search("kunhu"); // kunhu라는 키워드로 검색
  console.log(result);
}

main();

검색 옵션을 복잡하게 줄 수도 있지만, 이렇게 그냥 간단하게 텍스트만 날려서 검색을 돌릴 수도 있다.


그러면 잘 동작할 것이다.

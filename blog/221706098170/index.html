[Tcl] string 명령

  • 2019.11.13
  • ::
  • 1 min read
#Tcl
[원본 링크]

string 명령은 문자열을 처리하기 위한 명령어의 집합이다.
string** 명령종류 문자열값 추가인자?**

종류가 굉장히 다양한데, 간단하게 몇가지만 정리해본다.

length S
문자열의 길이를 반환한다.


compare L R
사전순으로 비교해서 같으면 0. L이 작으면 -1. R이 작으면 1.


equal L R
이건 compare보다 단순하다. 같으면 1.
다르면 0.


cat S1 S2 S3 ...
고양이다. 문자열을 다 연결해서 반환한다.


index S N
배열처럼 인덱스로 접근해서 문자를 뱉는다.


first s S
S에서 s가 나타나는 첫번째 부분 인덱스를 뱉는다.
없으면 -1

last s S
first와 반대로 뒤에서부터 찾는다. 그거말곤 같다.


이외에도 여러가지가 있으니, 문서를 참조하길 바란다.

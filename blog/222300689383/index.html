rollup은 group by의 부가기능으로, 소계를 구할 때 사용한다.
근데 뭐 그리 자주 쓸만한 기능은 아니다.

그룹해서 aggreate 연산을 한 것에 대해서 다시 aggreate 연산을 하고, 그걸 별개의 행으로 반환해주는 것이다.

가령, 다음과 같은 테이블과 데이터가 있을때

사람마다의 number 총합을 구한다면 이렇게 할 수 있을 것이다.

그런데 저기서 또 사람마다 sum한 값을 다시 합치고 싶으면 어떻게 해야 할까?
그룹을 한번 더 거는 방법도 있지만, rollup을 사용할 수도 있다.

rollup은 대체로 다음과 같이 그룹조건에 rollup을 씌우는 형태로 사용된다.

그럼 집계한 값에 대해서 다시 집계를 한 값을 새 행으로 추가해서 반환해준다.
집계값 이외의 나머지는 null로 채운다.

근데 이게 좀... 엑셀스러운 기능이라 일반적인 로직을 적용하기엔 애매한 부분이 보인다.
별개의 데이터를 억지로 행에 우겨넣는 것이기도 하고.

#Kubernetes
[원본 링크]

krew는 k8s의 플러그인 설치/관리용 보조도구다. 단순하고 직관적인 형태로 사용할 수 있는게 장점이다.




설치

Linux/MacOS의 기본 셸 환경에서는 다음과 같이 설치할 수 있다.

(
  set -x; cd "$(mktemp -d)" &&
  OS="$(uname | tr '[:upper:]' '[:lower:]')" &&
  ARCH="$(uname -m | sed -e 's/x86_64/amd64/' -e 's/\(arm\)\(64\)\?.*/\1\2/' -e 's/aarch64$/arm64/')" &&
  KREW="krew-${OS}_${ARCH}" &&
  curl -fsSLO "https://github.com/kubernetes-sigs/krew/releases/latest/download/${KREW}.tar.gz" &&
  tar zxvf "${KREW}.tar.gz" &&
  ./"${KREW}" install krew
)
export PATH="${KREW_ROOT:-$HOME/.krew}/bin:$PATH"

fish shell은 좀 다르다. 난 fish를 써서 이걸로 깔았다.

begin
  set -x; set temp_dir (mktemp -d); cd "$temp_dir" &&
  set OS (uname | tr '[:upper:]' '[:lower:]') &&
  set ARCH (uname -m | sed -e 's/x86_64/amd64/' -e 's/\(arm\)\(64\)\?.*/\1\2/' -e 's/aarch64$/arm64/') &&
  set KREW krew-$OS"_"$ARCH &&
  curl -fsSLO "https://github.com/kubernetes-sigs/krew/releases/latest/download/$KREW.tar.gz" &&
  tar zxvf $KREW.tar.gz &&
  ./$KREW install krew &&
  set -e KREW temp_dir &&
  cd -
end
set -gx PATH $PATH $HOME/.krew/bin

그러면 kubectl의 서브커맨드 형태로 krew를 호출할 수 있게 될 것이다.





플러그인 설치

directpv라는 도구를 다운받고 싶다면, 다음과 같이 install를 통해서 다운받을 수 있다.

kubectl krew install 플러그인

간단하다.

그리고 다운로드받은 확장 도구들은 마찬가지로 kubectl의 서브커맨드 형태로 사용이 가능하다.


설치한 플러그인들은 list로 확인 가능하다.

그 외에 버전 업그레이드는 update&upgrade, 설치 삭제는 uninstall 등으로 할 수 있다.

사용법은 별거 없다.

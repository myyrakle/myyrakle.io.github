Rust 코드를 문서화하는 것은 아주 간단하다.
현재 디렉토리 커맨드에서 cargo doc을 치면 된다.

그럼 target 폴더에 웹페이지 형식으로 된 문서를 알아서 생성해준다.

저기 프로젝트명 폴더 안을 까보면 index.html이 있다. 그걸 직접 열거나

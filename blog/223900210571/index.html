[Linux] Arch: pacman

  • 2025.06.15
  • ::
  • 2 min read
#Linux
[원본 링크]

pacman은 Arch Linux 계열의 배포판에서 사용하는 패키지 매니저다.

"pac"kage + "man"ager를 pacman으로 줄여서 말장난식으로 지은 이름인 것 같다.




패키지 설치 (Sy)

pacman에서 패키지를 설치할 때면, 보통 다음과 같은 형태로 사용한다.

sudo pacman -Sy 패키지명

add나 install 같은 명시적인 명령어가 없어서 좀 혼란스러울 수도 있다.

여기서 쓰는 플래그 S는 Sync 패키지를 설치하거나 재설치하는 명령 플래그다.
그리고 y는 로컬 패키지 DB를 리프레시한다는 의미다. y는 필수가 아니나, 보통 그냥 쓴다.




패키지 삭제 (-Rs)

패키지를 삭제할 때는 Rs 플래그를 넣어서 삭제할 수 있다.

sudo pacman -Rs vim

사실 여기서 삭제를 지시하는 플래그는 R이라서 s는 빼도 된다. 하지만 s를 줘야 그 종속성에 딸린 불필요한 종속성까지 함께 정리되므로, 항상 붙여서 쓰는게 권장된다.

만약 삭제하려는 패키지를 사용하는 패키지가 있다면 경고를 날리면서 실패한다.




패키지 검색 (-Ss)

패키지 검색은 s 플래그를 붙이는 것으로 수행할 수 있다.

pacman -Ss 검색어

그리고 특정 패키지의 상세 정보를 확인하려면, Qi 플래그를 쓰면 된다/



패키지 업데이트 & 업그레이드

전체 패키지를 업데이트&업그레이드하려면 다음 명령어를 쓰면 된다.

sudo pacman -Syu

이건 패키지 새 버전 정보를 업데이트만 하는게 아니라 실제로 업그레이드까지 한번에 다 한다.
apt-get update & apt-get upgrade와 같다고 보면 된다.

만약 업그레이드 없이 업데이트만 하고 싶다면 -Sy만 주면 되고

sudo pacman -Sy

업데이트 없이 업그레이드만 하고 싶다면 -Su만 주면 된다.

sudo pacman -Su

이 2가지를 합친게 -Syu인 셈이다.

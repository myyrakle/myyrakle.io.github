[Grafana] Grafana Cloud

  • 2024.07.30
  • ::
  • 1 min read
#Monitoring
[원본 링크]

Grafana Cloud는 Grafana 오픈소스들을 관리하는 회사에서 직접 만든 클라우드 서비스다.
Grafana 기반의 모니터링 리소스들을 말아서 일체형으로 제공한다.

근데 Grafana 특유의 성격이 어디가는건 아니라서, 대시보드 커스텀은 직접 해야한다.
원클릭으로 모든게 세팅되고 하는걸 바란다면 Sentry나 ELK나 그런거 쓰는게 낫다.




비용

당연히 비용은 있는데, 문턱이 좀 낮은 편이다.
신용카드 없이도 사용이 되고, 무료 사용량이 적당히 소규모에 쓸 정도는 된다.

돈내고 쓰더라도 어지간하면 프로로 충분할 것 같다. 어드밴스드는 24시간 고객대응같은 빠릿한 서비스를 원하는 사용자에게 맞다.

https://grafana.com/pricing/

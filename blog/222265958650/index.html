[Elixir] 맵

  • 2021.03.06
  • ::
  • 1 min read
#Elixir
[원본 링크]

엘릭서의 맵은 키:값 쌍으로 값을 관리할 수 있는 자료구조다.
구조체의 실제 내부 타입이기도 하다.

맵의 리터럴 표현은 다음과 같다.
%{ 키1 => 값1, 키2 => 값2 }

그리고 값의 접근은 다음과 같이 할 수 있다.
맵[키값]

아래는 그 간단한 예시다.

맵 제어용 함수들은 Map 모듈에 존재한다.
Map.put_new는 맵에 새 키와 값 쌍을 넣을 수 있는 함수다.


값 수정
특정 필드값만 변경한 채로 새 맵을 생성하고 싶다면, 다음과 같이 작성할 수 있다.
%{ 기존맵 | 바꿀키 => 바꿀값 }

그렇다.

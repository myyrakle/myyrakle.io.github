[Git] 기본 에디터 변경하기

  • 2023.12.10
  • ::
  • 1 min read
#Git
[원본 링크]

보통 별다른 설정을 하지 않으면 nano로 에디터가 구동되는 경우가 많은 것 같다.

나는 나노가 별로다.

vi로 에디터를 변경하고 싶으면 이렇게 하면 되고

git config --global core.editor "vi"

vscode로 쓰고 싶다면 이렇게 하면 된다.

git config --global core.editor "code --wait"

[AWS] AppSync: 시작

  • 2021.01.13
  • ::
  • 2 min read
#AWS
[원본 링크]

AppSync는 아마존에서 제공하는 GraphQL 전용 서비스다.

그냥 직접 GraphQL 프레임워크들을 땡겨다가 서버를 구축할 수도 있지만, 이걸 사용하면 귀찮은 관리사항들을 넘길 수 있어서 편리한 부분이 좀 있다고 한다.

일반적으로 AppSync에서는 스키마와 리졸버만 정의하고, 실제 데이터값들은 외부에서 가져오는 구조다.

그리고 VTL이라는 전용 템플릿 언어를 사용하는 점이 특징 중 하나다.

근데 이게 좀.. 호불호가 많이 갈린다.
소스를 짜서 올리는게 아니라 콘솔 페이지 자체에서 스키마와 리졸버, 연동 문제들을 다 해결하는 형식이고, 협업이나 형상관리 같은게 상당히 불편할 수 있기 때문이다.

그래서 아직은 프로덕션에 잘 쓰지 않는 듯한 모양새다.

한번 보자.
페이지에 들어가서 생성을 누른다.

[Unix] Fish Shell

  • 2024.07.02
  • ::
  • 2 min read
#Linux
[원본 링크]

fish shell은 Rust로 만들어진 인체공학적인 shell 도구다.
bash 같은 기본 셸들에 비해서 훨씬 사용성이 좋은게 특징이다.

Unix 계열, MacOS, Linux에서만 사용 가능하다. 윈도는 안된다.




설치 (Mac)

mac은 brew를 통해서 간편하게 설치할 수 있다.

brew install fish




설치 (Ubuntu)

우분투의 경우에는 apt를 통해서 설치할 수 있다.

sudo apt-add-repository ppa:fish-shell/release-3
sudo apt update 
sudo apt install fish -y

그리고 마찬가지로 fish 명령을 통해서 쉘에 진입할 수 있다.




하이라이팅

그냥 허옇게만 보여주는게 아니라, 명령에 대해서는 알록달록 하이라이팅을 해준다.




자동완성

가장 강력한 점은 자동완성이 지원된다는 것이다.
사용한 명령들을 기억하고, 그걸 기반으로 명령을 자동완성해준다.

방향키 오른쪽을 누르면 자동완성을 사용하게 된다.




도움말

man 명령을 사용하면 다른 명령에 대한 도움말을 얻을 수 있다.
예를 들어, df의 사용법을 알고 싶다면 이렇게 하면 된다.


이외에도 자잘한 보조기능들이나 특수화된 셸 문법들이 좀 있지만, 크게는 이 정도가 대표적인 차별점이라고 할 수 있다. 이 정도만 소개하고 넘어가겟다.




기본 셸로 만들기

fish 실행파일의 경로를 확인한다.

저걸 chsh의 인자로 넣어주면 기본 셸이 변경되는 것이다.

chsh -s $(which fish)

sudo로 때리면 루트에 대해서만 적용이 된다는 점에 유의한다. 현재 유저에 대해서 적용하려면 sudo 없이 사용해야 한다.

그러고 컴퓨터를 재부팅하고 셸을 켜면

기본으로 켜질 것이다.

macOS에서는 /etc/shells라는 파일에 경로를 추가해줘야 한다.

VSCode 같은데서도 어렵지 않게 기본 셸로 변경할 수 있다.

잘 된다.

  • 2025.06.23
  • ::

#Github
[원본 링크]

Github Copilot도 Cursor, windsurf, Cline 같은 에이전트 프로그래밍 기능을 지원한다.

아직 정식 기능은 아니고 프리뷰 모드다.
현재 시점에서는 Copilot Pro, Prod+, Copilot Enterprise에서 모두 사용 가능하다.

사용법은 간단하다.

그냥 Copilot Chat 기능에서 확장된 형태고, 채팅할때 에이전트 모드로 대화하면 된다.


요구사항대로 요청을 하면


cursor 등과 마찬가지로 코드를 생성해서 선택권을 준다.
cline보다 사용성이 더 좋은 것 같기도 하다.




Rules 관리

이것도 프롬프트 규칙을 어느 정도 정의해줄 수 있다.
.github/copilot-instructions.md 경로에 텍스트로 적절히 적어주면 된다.

그러면 채팅을 칠때마다 알아서 읽어서 참고한다.

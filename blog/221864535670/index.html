[Rust] serde: JSON

  • 2020.03.21
  • ::
  • 2 min read
#Rust
[원본 링크]

serde는 다양한 데이터형식들을 직렬화/역직렬화하게 도와주는 멋진 라이브러리다.
JSON, BSON, YAML, TOML, URL 등등 대부분의 형식을 지원한다.

우선은 이중에서 가장 뜨거운 포맷인 JSON을 먼저 다뤄보겠다.

먼저 serde_json을 종속성에 추가한다.

그럼 밑준비는 끝이다.



문자열->객체

json 문자열에서 json 객체를 뽑아내는 것부터 해보자.

json 규격을 충족하는 간단한 형태의 문자열을 만들어봤다.

저걸 파싱하려면 from_str 등의 함수를 사용하면 된다.

Value는 재귀적으로 정의된 열거 타입이다.
그래서 모든 json 객체는 Value 타입이기도 하다.

여튼, 추출된 Value 객체에서 요소를 가져오려면, 그냥 해당 요소의 이름으로 인덱싱을 하면 된다.
그럼 다시 해당 요소를 Value로 감싸서 반환하는 것이다.

근데 전부 Value로만 퉁쳐서는 제대로된 처리가 불가능하다.
Value라는 껍질을 벗겨서 기본타입들로 바꾸려면 대응되는 as_... 메서드들을 사용하면 된다. 이건 Option을 반환한다.
가령 as_i64는 해당 Value가 숫자일 경우엔 Some(값)을, 아니면 None을 반환하는 것이다.

저대로 돌리면 잘 돌 것이다.

전체 소스는 이렇다.

fn main() 
{
    //json 문자열
    let json = r#"
    {
        "name":"홍길동",
        "age":300,
        "hobby":["영화감상", "등산", "해적질"] 
    }
    "#;

    use serde_json::Value;

    //문자열을 파싱해 json 객체 Value를 생성
    let result:Value = serde_json::from_str(json)
        .expect("파싱 실패");

<br>

    let name:Value = result["name"].clone();
    println!("name:{}", name);

    let age:i64 = result["age"].as_i64()
        .expect("실패");
    println!("age:{}", age);

    println!("hobby:{}", result["hobby"]);
}


객체->문자열

이번엔 json 객체를 생성해서 그걸 문자열로 바꿔보자.
json 객체를 만드는 가장 손쉬운 방법은 json! 매크로를 사용하는 것이다.
이 매크로 함수는 json 문법을 그래도 써넣어서 Value 객체를 생성할 수 있다.

이렇게 말이다.

기존 Value 객체에 새 요소를 집어넣고 싶다면, 그냥 인덱싱을 시도하면서 새 Value 값을 끼워넣으면 된다.



구조체를 이용한 직렬/역직렬화

근데 serde에서는 저러한 json에 종속적이고 불편할 수 있는 접근 없이. 구조체를 정의해서 그걸 통해 곧장 json 객체를 파싱하거나 생성할 수 있다.
그러려면 serde를 의존성으로 추가해줘야 한다. derive를 기능으로 넣어주고.

그럼 이제 해보자

사용할 json의 형식대로 구조체를 정의한다.
그리고 Deserialize와 Serialize를 derive로 파생시킨다.

그럼 이것만으로도 각 변환이 자유자재로 동작하게 된다.


출력용 메서드도 간단하게 정의했다.

그럼 문자열->객체로의 변환을 수행해보자
이렇게 json 텍스트가 있다면

파싱을 시도할 때 Value 타입을 거치지 않고 바로 해당 구조체로 받을 수가 있다!
해당 json 데이터가 구조체에 완전히 들어맞는다는 가정 하에서 말이다.


객체를 만들어서 json 텍스트로 변환하는 것도 간단하다. 그냥 serde_json::to_string 메서드에 참조로 전달해주면 Result로 감싸서 반환해준다.

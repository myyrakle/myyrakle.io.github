[Tcl] 분기

  • 2019.11.13

#Tcl
[원본 링크]

Tcl의 분기에는 if-else, switch 등이 있다.

if-else문
기본적인 분기문이다. 아래의 구조를 가진다/
if {조건식} {
**    ...**
} else {
**    ...**
}

아래는 간단한 예다.

짜증나는건 저기서 else는 개행을 하면 안된다는 것이다. if 닫는 중괄호에 안붙이면 못읽는다.

사실 조건식과 몸체 사이엔 then을 넣을 수도 있다. 안넣어도 상관은 없다.

중첩분기를 구현하려면, elseif 키워드를 사용한다.


switch문
다중분기문이다.
대충 아래의 구조를 가진다.

switch 플래그 -- 인자값 {
**    패턴1 {표현식}**
**    패턴2 {표현식} ...**
**    default {표현식}**
}

플래그로는 아래의 3가지가 있는데. 골라쓰면 된다.
-exact :정확한 패턴만 비교
-glob :glob 형태의 패턴매칭
-regexp :정규표현식

그렇다.

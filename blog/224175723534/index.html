IngressлҠ” HTTP мҲҳмӨҖмқҳ лқјмҡ°нҢ…мқ„ м ңм–ҙн•  мҲҳ мһҲкІҢ н•ҙмЈјлҠ” кі кёү м»ҙнҸ¬л„ҢнҠёлӢӨ.
API Gateway нҢЁн„ҙм—җ лҢҖн•ң к·ңкІ©мқ„ мҝ лІ„ мһҗмІҙм—җм„ң н‘ңмӨҖнҷ”н•ҙм„ң кҙҖлҰ¬н•ҳлҠ” кІғмқҙлӢӨ.

IngressлҘј мӢӨм ңлЎң м—°кІ°н•ҳлҠ” кІғмқҖ Ingress ControllerлқјлҠ” лі„лҸ„ мӢңмҠӨн…ңмқҳ мұ…мһ„мқҙлӢӨ.
Ingress Controllerмқҳ л„җлҰ¬ м“°мқҙлҠ” кө¬нҳ„мІҙлҠ” nginx ingress controller(мӮ¬л§қ), istio ingress, traefik ingress controller, envoy л“ұмқҙ мһҲлӢӨ.

к·јлҚ° м–ҙм§ём„ңмқём§Җ мҝ лІ„л„ӨнӢ°мҠӨ к·ёлЈ№м—җм„ңлҠ” мқҙкұё нҶөм§ёлЎң лҚҳм§Җкі  кІҢмқҙнҠёмӣЁмқҙлқјлҠ” мғҲлЎңмҡҙ кё°лҠҘмңјлЎң лҢҖмІҙн•ҳл ӨлҠ” мӢңлҸ„лҘј н•ҳкі  мһҲлҠ” кІғ к°ҷлӢӨ.




кё°ліё кі„мёө (Traefik)

к°ңлі„ Ingress Controllerл“Өмқҳ м„Өм№ҳ л°©лІ•м—җ лҢҖн•ҙ лӢӨлЈЁм§„ м•ҠлҠ”лӢӨ.
м—¬кё°м„ңлҠ” Traefik Ingress Controllerк°Җ м„Өм№ҳлҗҳм–ҙмһҲлӢӨкі  к°Җм •н•ңлӢӨ.

л§Ңм•Ҫ Traefik Ingress Controllerк°Җ мһҳ к№”л ӨмһҲлӢӨл©ҙ, лӢӨмқҢкіј к°ҷмқҖ мӢқмңјлЎң лҰ¬мҶҢмҠӨк°Җ мЎ°нҡҢлҗ  кІғмқҙлӢӨ.

deployment н•ҳлӮҳм—җ service н•ҳлӮҳк°Җ мһҲлӢӨ.

мӮ¬мӢӨ мқҙ Ingress ControllerлқјлҠ” кІғмқҙ к·ёл ҮкІҢ нҠ№лі„н•ң кІғмқҖ м•„лӢҲлӢӨ.
ServiceмҷҖ Deployment мҢҚмңјлЎң мқҙлЈЁм–ҙм§„ Gateway м„ңлІ„к°Җ мһҲкі , мқҙкІҢ к°ңлі„ APIл“Өмқ„ лқјмҡ°нҢ…н•ҙмӨ„ лҝҗмқҙлӢӨ.

Ingress Controllerмқҳ м—”л“ңнҸ¬мқёнҠёлҠ” ServiceлҘј нҶөн•ҙ м ңкіөлҗҳлҠ”лҚ°, лӢӨлҘё Serviceл“Өкіј лҸҷмқјн•ҳлӢӨ.
м•Ңм•„м„ң External IPлҘј лҡ«л“  н•ҙм„ң к°ңл°©н•ҙм•ј н•ңлӢӨ.




Ingress м¶”к°Җн•ҙліҙкё° (ex: Traefik)

IngressлҘј м¶”к°Җн•ҳл Өл©ҙ лЁјм Җ л¶ҷмқј м„ңл№„мҠӨк°Җ мһҲм–ҙм•ј н•ңлӢӨ. м—¬лҹ¬к°ңмқҳ м„ңл№„мҠӨлҘј лӢЁмқј лЎңл“ңл°ёлҹ°м„ңм—җ лқјмҡ°нҢ…н•ҳлҠ” кІғмқҙкё° л•Ңл¬ёмқҙлӢӨ.

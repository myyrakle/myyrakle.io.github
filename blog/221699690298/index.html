[D] 기본 타입

  • 2019.11.06
  • ::
  • 1 min read
#D
[원본 링크]

D의 기본 타입은 아래와 같다.


**논리 **(디폴트값 false)
bool (1바이트) (true or false)


**정수 **(디폴트값 0)
byte (1바이트) (부호있음)
ubyte (1바이트) (부호없음)
short (2바이트) (부호있음)
ushort (2바이트) (부호없음)
int (4바이트) (부호있음)
uint (4바이트) (부호없음)
long (8바이트) (부호있음)
ulong (8바이트) (부호없음)

작은 타입->큰 타입, 부호 변환은 자유롭지만,
큰 타입->작은 타입으로의 자동 변환은 불가하다.


**실수 ****(***디폴트값 .nan)
float (4바이트)
double (8바이트)
real (cpu 최대 부동소수점. x86는 100바이트)


**허수 ***(디폴트값 .nan x 1.0i)
ifloat
idouble
ireal


**복소수 **(디폴트값 .nan + (.nan x 1.0i))
cfloat
cdouble
creal


문자** **
char (1바이트) (UTF-8)
wchar (2바이트) (UTF-16)
dchar (4바이트) (UTF-32)


문자열
string (char 배열)
wstring (wchar 배열)
dstring (dchar 배열)


기타
void (사이즈 없음)
반환타입 표현에만 사용


참조
https://dlang.org/spec/type.html

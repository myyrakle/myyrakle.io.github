[Elm] 레코드

  • 2020.03.10
  • ::
  • 1 min read
#Elm
[원본 링크]

레코드는 튜플과 유사한 복합 타입이다. 마찬가지로 여러가지 타입의 값들을 하나로 묶을 수 있다.
하지만 이건 튜플과 다르게 각 요소에 이름이 붙는다.
그래서 이름을 통해 접근할 수 있어, 사용이 더 용이하다.

다른 언어에서의 구조체 등과 동등하다.

일단 레코드는, 중괄호로 감싸고, 쉼표로 구분된 변수 선언의 목록으로 표현된다.
{ 필드1=값1, 필드2=값2, ... }

그리고 레코드의 필드는 지정된 필드명으로 접근이 가능하다.



레코드의 갱신

레코드도 일반 변수와 마찬가지로 변경이 불가능하다.

또다른 값의 상태를 표현하려면 레코드를 재생성해야 하는데...
여기엔 괜찮은 전용 표현식이 있다.
아래의 형식을 쓰면, 명시된 필드만 바뀌고 다른건 그대로인 레코드 객체가 복사되어 반환된다.
{ 레코드값 | 바꿀필드... }



타입명 부여

근데 레코드는 리스트와 달리, 타입을 항상 그대로 써서 명시하기가 영... 불편하다.
레코드를 제대로 쓰려면 type alias로 이름을 붙여주는게 좋다.

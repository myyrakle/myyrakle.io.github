[node.js] node.js 시작하기

  • 2019.02.24
  • ::
  • 1 min read
#Javascript
[원본 링크]

일단 사이트에 들어가서 인스톨러를 설치한다.
https://nodejs.org/en/
Current보다는 안정적인 LTS가 좋다.

그리고 그냥 예스맨이 되면 된다.

다 설치하고 나면 cmd를 켜고 node를 쳐본다.

이렇게 대화형 인터프리터가 실행되면 성공이다.

자바스크립트 문법으로 표현식을 써넣고 엔터를 치면 반환값을 보여준다,
파이썬 그거랑 똑같다.

인터프리터를 종료하려면 process.exit()를 쓰면 된다.
process는 기본적으로 존재하는 전역 객체 중 하나다.



근데 뭐 이런식으로는 제대로 된 개발 작업을 할 수가 없다.
제대로 쓰는 방법은 커맨드에서 gcc로 컴파일할때처럼 명령인자를 보내는 것이다.

이렇게 소스파일을 작성하고

node 소스파일...
형식으로 호출하면 된다.
저 console도 이름 그대로 콘솔을 다루는 기본 전역 객체다.

근데 계속 이렇게 메모장으로 작업을 할순 없지 않은가?
제대로 작업을 하려면 에디터를 써야하는데 나는 Visual Studio Code를 사용하므로 이를 기준으로 기술하겠다.
Visual Studio에도 node.js 확장이 있는 것 같던데 써보질 않아서 어떤지는 모르겠다.

일단 문법 체크를 위하여 node.js 확장을 설치해줘야 한다.
몇개 더 있는데 난 저걸로 골랐다.

작업 폴더를 잘 만들고
launch.json을 열어서 아래와 같이 만든다.

{
    "version": "0.2.0",
    "configurations": [
        {
            "type": "node",
            "request": "launch",
            "name": "프로그램 시작",
            "program": "${file}",
            "cwd": "${workspaceRoot}"
        },
        {
            "type": "node",
            "request": "attach",
            "name": "프로세스에 연결",
            "port": 5858
        }
    ]
}

잘 저장하고, 이제 컨트롤+f5를 눌러서 실행이 잘 되면

세팅 완료다.

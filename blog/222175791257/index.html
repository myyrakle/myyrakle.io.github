[PostgreSQL] copyлҘј мқҙмҡ©н•ң лҢҖлҹү мӮҪмһ… (bulk insert)

#PostgreSQL
[мӣҗліё л§ҒнҒ¬]

м•„мЈј л°©лҢҖн•ң лҚ°мқҙн„°лҘј лӢӨлЈЁлҠ” мӢңмҠӨн…ңмқҳ кІҪмҡ°м—җлҠ”, лҢҖлҹүмңјлЎң лҚ°мқҙн„°лҘј мӮҪмһ…н•ҳлҠ” кІҪмҡ° лҳҗн•ң л§ҺлӢӨ.

к·ёлҹҙл•ҢлҠ” н•ӯмғҒ м—„мІӯлӮң мӢңк°„мқ„ мһЎм•„лЁ№лҠ”лҚ°... к·ё мҶҢмҡ”мӢңк°„мқ„ мўҖ мӨ„м—¬ліј л°©лІ•мқҙ мһҲлӢӨ.
copy лӘ…л №м–ҙлҘј мӮ¬мҡ©н•ҳлҠ” кІғмқҙлӢӨ.

мқҙкұё мӮ¬мҡ©н•ҳл©ҙ мӮҪмһ…н•  лҚ°мқҙн„°л“Өмқ„ н…ҚмҠӨнҠёнҢҢмқјм—җлӢӨ м ҖмһҘн•ҙлҶ“кі , к·ёкұё к·ёлҢҖлЎң к°Җм ёмҳ¬ мҲҳ мһҲлӢӨ.
лҚ°мқҙн„°нҢҢмқјмқҳ нҳ•мӢқмқҖ кё°ліём ҒмңјлЎң н•ң мӨ„лӢ№ н•ң лҚ°мқҙн„°лҘј лң»н•ҳл©°, м»¬лҹјмқ„ кө¬л¶„н•ҳлҠ” кёҖмһҗлҠ” м§Ғм ‘ м§Җм •н•  мҲҳ мһҲлӢӨ.

copyлҠ” м•„лһҳмқҳ кө¬мЎ°лҘј к°Җм§„лӢӨ.

copy н…Ңмқҙлё”лӘ…(м»¬лҹјлӘ…, ...) from '...нҢҢмқјкІҪлЎң' with delimiter 'м»¬лҹј кө¬л¶„мһҗ'

н…ҢмҠӨнҠён•  н…Ңмқҙлё”мқҖ мқҙл ҮкІҢ мғқкІјлӢӨ.

лі„кұ°м—ҶлӢӨ.

н•ңлІҲ н…ҢмҠӨнҠёлҚ°мқҙн„°лҘј л§Ңл“Өм–ҙм„ң лҸҢл Өліҙмһҗ
лӮҳлҠ” м•„лһҳмҷҖ к°ҷмқҖ мһҗлҸҷнҷ” мҪ”л“ңлЎң мҝјлҰ¬мҷҖ лҚ°мқҙн„°нҢҢмқјмқ„ мғқм„ұн–ҲлӢӨ.
лҚ°мқҙн„°лҠ” 100л§Ңк°ңкі  мҪ”л“ңлҠ” RustлӢӨ.

use rand::Rng;

fn main() {
    let values_nums: Vec<String> = (1..=10000000)
        .map(|_| {
            let mut rander = rand::thread_rng();
            let num: i64 = rander.gen();
            format!(r#"('NO.{}', {})"#, num, num)
        })
        .collect();

    let text_nums: Vec<String> = (1..=10000000)
        .map(|_| {
            let mut rander = rand::thread_rng();
            let num: i64 = rander.gen();
            format!(r#"'NO.{}', {} "#, num, num)
        })
        .collect();

    let values = values_nums.join(", ");
    let query = format!(
        r#"
        explain analyze 
        insert into tb_test(string, num)
        values {}
    "#,
        values
    );

    let text = text_nums.join("\n");

    std::fs::write("query.sql", query).expect("м“°кё° мӢӨнҢЁ");
    std::fs::write("just_text.txt", text).expect("м“°кё° мӢӨнҢЁ");
}

к·ёлһҳм„ң мӮҪмһ… мҝјлҰ¬лҠ” мқҙлҹ°мӢқмңјлЎң л§Ңл“Өкі 

copyн•  лҚ°мқҙн„°лҠ” мқҙл ҮкІҢ л§Ңл“Өм—ҲлӢӨ.

мқјлӢЁ кё°мЎҙмқҳ л°©мӢқлҢҖлЎң к·ёлғҘ insertлҘј мӮ¬мҡ©н•ҙліҙмһҗ

л¶„м„қмӢңк°„ м•Ҫ м•Ҫ 3мҙҲ + мӢӨн–үмӢңк°„ м•Ҫ 194мҙҲк°Җ кұёл ёлӢӨ.
кұ°мқҳ 197мҙҲк°Җ кұёлҰ° м…ҲмқҙлӢӨ.

к·ёлҰ¬кі  copyлҘј мӮ¬мҡ©н•ҙлҙӨлӢӨ.
мңҲлҸ„мҡ°мқҳ кІҪмҡ°м—җлҠ” лҚ°мқҙн„°нҢҢмқјмқ„ public кІҪлЎңм—җ л‘¬м•јл§Ң мқҪм–ҙмҳ¬ мҲҳ мһҲмңјлӢҲ мң мқҳл°”лһҖлӢӨ.

copyлҠ” мҝјлҰ¬лҸ„ м•„лӢҲкі  мӢӨн–үкі„нҡҚк°ҷмқҖкІҢ м—Ҷм–ҙм„ң explainмқҖ лӘ»н–ҲлҠ”лҚ°, мҠӨнҶұмӣҢм№ҳлЎң м§Ғм ‘ мһ¬ліё кІ°кіјлЎңлҠ” 185мҙҲ м •лҸ„к°Җ кұёл ёлӢӨ.

к·ёлҰ¬кі  лҚ°мқҙн„°мқҳ нҒ¬кё°к°Җ м»Өм§ҲмҲҳлЎқ copyмқҳ м„ұлҠҘмқҙ лҚ” мўӢм•„м§„лӢӨкі  н•ңлӢӨ. мһЎлӢӨн•ң мІҙнҒ¬к°ҷмқҖкІҢ м—Ҷмқҙ л¬ҙмһ‘м • м§‘м–ҙл„Јкё°л§Ң н•ҳкё° л•Ңл¬ёмқҙлһҖлӢӨ. мӢӨн–үкі„нҡҚ кі„мӮ°лҸ„ м•Ҳн•ҳкі .

к·јлҚ° лӯҗ.. мқјл°ҳм Ғмқё мғҒнҷ©м—җлҠ” нҷңмҡ©н•  мқјмқҙ л”ұнһҲ м—ҶлҠ”кІҢ, мҲҳк°Җ м—„мІӯ л§ҺмқҖкІҢ м•„лӢҲл©ҙ к·ёл ҮкІҢ л§ү л№ лҘё кІғлҸ„ м•„лӢҲлӢӨ.
лҸҢл ӨліҙлӢҲк№Ң 10л§Ңк°ңк№Ңм§ҖлҠ” м„ұлҠҘмқҙ кұ°мқҳ л№„мҠ·н•ҳкі , 100л§Ңк°ңл¶Җн„° м„ұлҠҘмқҙ мўҖ лІҢм–ҙмЎҢлӢӨ.

лҚ°мқҙн„°к°Җ кұ°мқҳ м–өлӢЁмң„лЎң к°Җл©ҙ нҷ• лІҢм–ҙм§„лӢӨкі  н•ңлӢӨ.

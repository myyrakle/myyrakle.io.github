[MongoDB] 개괄

  • 2019.12.23
  • ::
  • 1 min read
#MongoDB
[원본 링크]

몽고DB는 2009년에 개발된 NoSQL 데이터베이스 시스템이다.
NoSQL 중에서도 가장 유명하고 대중적인 녀석이다.

그럼 NoSQL은 뭐냐? 기존의 데이터베이스 시스템 구조인, 관계형 방식을 사용하지 않는다는 말이다.
테이블 생성하고 정규화해서 분리하고 참조하고... 그런 구조 말이다.
여튼 NoSQL은 기존 SQL의 복잡한 구조에서 유발하는 성능과 생산성 저하를 개선하고자 나온 녀석들이다.

분류를 NoSQL이라곤 그냥 부르곤 하지만, 정확하겐 "Document 지향"이라는 시스템을 가지고 있다.
그냥 JSON 방식으로 데이터를 저장하고 관리하는 거다.

그리고 기존 DBMS들이 SQL이라는 규격화된 스크립트 언어를 사용한 것과 다르게, 자바스크립트를 스크립트로 사용하는 게 특징이다.
때문에 자바스크립트를 사전에 알고있는 편이 좋다...

암튼 여러 대기업에서도 주요 백엔드 시스템으로 사용하기도 하고, 유망한 것은 분명하다.

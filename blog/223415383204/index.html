[Haskell] 설치하기 (in Ubuntu)

  • 2024.04.14
  • ::
  • 1 min read
#Haskell
[원본 링크]

커뮤니티에서는 ghcup이라는 통합 관리자로 설치하는 것을 권장하는 것 같다.

아래 사전 종속성이 없다면 먼저 설치해준다.

sudo apt-get install -y build-essential curl libffi-dev libffi8 libgmp-dev libgmp10 libncurses-dev libncurses5 libtinfo5

그리고 다음 스크립트를 실행해서 설치를 진행한다.

curl --proto '=https' --tlsv1.2 -sSf https://get-ghcup.haskell.org | sh

그럼 대화형으로 설치 페이지가 나타나는데, 대충 디폴트값으로 다 무지성 설치해도 된다.


다 설치됐다면 이렇게 뜬다.
터미널 세션을 새로 열면 그때부터 사용할 수 있다.

ghci는 대화형 인터프리터고

ghc는 컴파일러다.

이걸로 컴파일해서 실행하면 된다.

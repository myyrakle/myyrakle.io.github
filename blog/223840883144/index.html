#Github
[원본 링크]

코파일럿은 Github 환경에서의 코드 리뷰 기능을 제공한다.
당연히 코파일럿을 사용중이어야 적용 가능하다.

Pro부터 자유롭게 사용할 수 있으나, 사용랑에 제한은 존재한다.


여기서 "프리미엄 요청"이란 것을 한번씩 소모한다.
그래서 너무 남발하지 않는 것이 중요하다.




사용법

사용법은 간단하다.
그냥 pull request에서

요청을 누르면 된다.


그럼 이런식으로 이것저것 짚어준다.
언어는 영어만 지원하는 것 같다.




가이드라인 설정

코드 리뷰에 대한 규칙과 가이드라인을 어느 정도 사전에 정의해서 적용할 수도 있다.

다만 이건 개인 플랜에서는 아직 제공하지 않고, Copilot Enterprise를 쓰는 주요 고객들에게만 제공된다.
뭐 대단한거라고 이렇게까지 막아놓지...?

VecDeque[T]

양방향 큐를 구현했다.
내부 구현은 배열 기반의 원형 버퍼다.

앞뒤로 빠르게 추가할 수 있으면서, 배열의 빠른 성능도 가져갈 수 있다.




문서화 보완

문서화 부족했던 부분들 좀 채우고, Assertion 기반의 예제코드들을 짤막하게라도 실어놨다.





Assertion 함수 추가

테스트 등에 사용할 수 있는 assertion 함수들과

패닉 함수 몇가지를 정의했다.

Todo, Unreachable, Unimplemented는 단순한 패닉의 래핑으로, 문맥상 특수한 의미를 나타내기 위한 용도로 사용한다.




Mutex

뮤텍스도 구현했다.
go의 느슨한 형태가 아니라, 타입을 감싸놓은 래퍼 타입이라서 제약을 더욱 강하게 걸어 쓸 수 있다.




Clone 트레잇 구현

대부분의 타입에서 Clone을 통해 깊은 복사 효과를 볼 수 있다.




Eq 트레잇 구현

이제 대부분의 타입에서 Eq 메서드를 통해 깊은 비교를 수행할 수 있다.




AsRef 트레잇 추가

대부분의 타입에 구현된 AsRef 트레잇을 통해 포인터 값을 간편하게 가져올 수 있다.

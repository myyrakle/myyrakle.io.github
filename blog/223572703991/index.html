[Rust] 참조: Cow

  • 2024.09.05
  • ::
  • 2 min read
#Rust
[원본 링크]

Cow는 Clone on Write(Copy on Write)에 대한 동작을 제공하는 enum 기반 구현체다.

일반적으로 불변 참조에 대한 값을 들고 있으나, write를 시도하면 복제를 시도한 후에 변경된 복제본을 반환하는 형태의 동작을 제공한다.

기본적으로는 이런 형태를 가진 enum이다.
평상시에는 Borrowed로 참조값만을 들고 있다가, 수정이 발생하면 복제본을 만들어서 반환하는 것이다.

기본적으로 크기가 클 수 있는 타입에 대해서 사용하기 용이하다.
C++도 C++11 이전에는 string이 Cow 기반 구현체였던 적이 있다.

한번 직접 Cow의 요건을 만족하는 구현체를 만들어서 동작 방식을 확인해보자.
내 경우에는 간단한 문자열 래퍼를 만들었다.

출력과 재설정 메서드 정도만 채워넣고


ToOwned 트레잇을 구현한다. Cow에 넣으려면 이게 필수조건이다.
그리고 복제를 확인하기 위해 로그도 한줄 찍었다.


그리고 MyString->Cow로의 변환 동작을 구현한다.
Borrowed로 참조로 초기 생성을 하도록 했다. 그러지 않으면 Cow를 쓰는 의미가 없다.

그러고 나면 다음과 같이 사용할 수 있다.

Cow::from으로 참조 기반 객체를 생성하고, 그냥 불변 동작으로 쓸 때는 대충 쓰다가, 수정 동작이 필요하면 to_mut로 mutable 참조를 받아서 사용하는 것이다.
to_mut 시점에 생성되는 mutable 참조는 원본 객체와는 무관하며, to_owned로 복제된 새 객체일 뿐이다.


실제로 돌려봐도 의도한대로 찍힐 것이다.

아래는 전체 코드다.

use std::borrow::Cow;

struct MyString(pub String);

impl MyString {
    fn show(&self) {
        println!("{}", self.0);
    }

    fn reset(&mut self, s: &str) {
        self.0.clear();
        self.0.push_str(s);
    }
}

impl ToOwned for MyString {
    type Owned = MyString;
    fn to_owned(&self) -> MyString {
        println!("복제됨");
        MyString(self.0.clone())
    }
}

impl<'a> From<&'a MyString> for Cow<'a, MyString> {
    fn from(s: &'a MyString) -> Cow<'a, MyString> {
        Cow::Borrowed(s)
    }
}

fn main() {
    let big_str = MyString("very big string".to_string());

    let input = Cow::from(&big_str);

    input.show();
    println!("big_str: {:?}", big_str.0);

    let mut input = Cow::from(&big_str);

    {
        let big_str_2 = input.to_mut();
        big_str_2.reset("reset string");

        println!("big_str_2: {:?}", big_str_2.0);
    }

    input.show();
    println!("big_str: {:?}", big_str.0);
}

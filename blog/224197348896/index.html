#LLM
[원본 링크]

codex는 openai에서 만든 claude code의 카피캣이다. GPT 모델에 채팅 CLI 기반으로 개발 등 다양한 작업들을 처리할 수 있게 해준다.

짝퉁으로 시작했지만, 요즘엔 대체로 claude code와 비슷하거나 그 다음가는 정도라는 평가다.



설치

npm으로 설치할 수 있다.

npm i -g @openai/codex

설치하면 codex라는 이름으로 실행이 가능해야 한다.
nodejs 설치방식에 따라서 저게 설치된 경로가 path에 없을 수도 있다.




사용법

일단 처음 실행하면 인증부터 요구한다.

chatgpt 정액제와 api key 종량제 2가지를 기반으로 사용할 수 있다. 나는 정액제였다.


연동이 되면 즉시 채팅 모드로 진입한다. 떠들면서 만져보면 된다.




IDE 통합 (VSCode)

VSCode를 비롯한 개발 도구와도 잘 통합된다.
개인적으로는, 이런 부분에서는 클로드보다 나은거같다.
클로드는 인증 세션 관리가 개판이라서 vscode로 붙일때 인증을 또 요구하고 그랬는데, 이건 그냥 되더라.

확장만 깔면


왼쪽 패널에 채팅창이 생긴다.
기본적인 ux는 다른 에이전트 패널들과 98% 일치한다. 서로서로 다 베껴대니...


모델도 이렇게 선택할 수 있고


코드 짜는것도 적당히 잘 한다.




체감 성능

솔직히 말해서, 아직 클로드에 비교하면 좀 딸린다.

의도를 파악하는 능력도 좀 부족하고, IDE와의 통합 수준도 아직 딸린다. 지금 열린 코드를 자동으로 참조하는게 안된다. (VSCode)

일반적인 상황에서는 클로드 코드와 큰 차이를 못느낄 수도 있으나, 문제해결력이나 사소한 부분들에서 찐빠를 내는 경우가 꽤 잦았다.

아직 지피티는 좀 먼거같다.

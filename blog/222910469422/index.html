#Cloudflare
[원본 링크]

Pages는 cloudflare의 정적호스팅 서비스다.

기존에 github page 공짜로 쓰던걸 도메인 세팅을 바꿔야하는데, 얘네들이 좀 세팅이 번거로워서 그냥 cloudflare로 이전하기로 했다.

클플도 무료 사용량을 둔 채로 제공해준다.
일일 10만 리퀘스트다.

생성버튼 누르고

기존 레포지토리에서 바로 가져올 거라면 convert to git을, 그냥 파일업로드를 할거라면 아래 버튼을 누른다.
내 경우엔 전자다.


깃허브 연결해준다음에


레포지토리 고르고 셋업버튼을 누른다.


빌드작업이 필요하다면 빌드커맨드로 빌드작업을 수행한 다음에 그 dist 경로를 넣어서 올리게 해줄 수 있다.
빌드환경에는 npm이 기본적으로 제공된다.

내 경우엔 해당 레포에 결과물이 다 있어서 빌드 없이 루트경로로만 제공했다.


그렇게 완료버튼 누르면 프로세스가 돌고


배포가 될 것이다.


짜잔



cloudflare 자체도메인 붙이기

cloudflare에서 구입한 도메인을 갖고있다면, 붙이는건 몹시도 간단하다.

원하는 도메인명 써주고


체크버튼만 눌러주면 된다.


몹시 잘된다.

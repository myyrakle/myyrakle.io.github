#ML
[мӣҗліё л§ҒнҒ¬]

Stable DiffusionмқҖ мқҙлҜём§Җ мғқм„ұ лӘЁлҚё мӨ‘м—җм„ңлҸ„ к°ҖмһҘ н•өмӢ¬м Ғмқё мң„м№ҳлҘј м°Ём§Җн•ҳкі  мһҲлҠ” мҳӨн”ҲмҶҢмҠӨ лӘЁлҚёмқҙлӢӨ.
2022л…„м—җ мІҳмқҢ лӮҳмҷ”кі , Stability AIлқјлҠ” кё°м—…м—җм„ң л§Ңл“Өм—ҲлӢӨ.

лҢҖкё°м—…л“Өмқҳ нҸҗмҮ„нҳ• лӘЁлҚёмқ„ м ңмҷён•ҳкі  мҳӨн”ҲмҶҢмҠӨ лӘЁлҚё мӨ‘м—җм„ңлҠ” м••лҸ„м Ғмқё мң„м№ҳлҘј м°Ём§Җн•ҳкі  мһҲлӢӨ. м„ұлҠҘлҸ„ к°ҖмһҘ мўӢкі , мҶҚлҸ„лҸ„ л№ лҘҙлӢӨ.

лӢЁ, л¬ҙлЈҢмқҙкёҙ н•ңлҚ°, мқҙкұҙ л№„мғҒм—…мҡ©м—җ н•ңм •н•ң кІғмқҙлӢӨ. мғҒм—…мҡ©мңјлЎң мӮ¬мҡ©н•ҳл Өл©ҙ л№„мҡ©мқ„ лӮҙм•ј н•ңлӢӨ.




лҸҷмһ‘ мӣҗлҰ¬

Stable Diffusionмқҳ нҒ¬кІҢ 3к°Җм§Җ л¶Җл¶„мңјлЎң лӮҳлүңлӢӨ.
мӢӨн–ү мҲңм„ңлҢҖлЎң CLIP, UNet, VAE(Variational Auto Encoder)лӢӨ.

мІ«лІҲм§ё, CLIP(Contrastive Language-Image Pre-training)мқҖ мң м Җмқҳ н…ҚмҠӨнҠёлҘј л°ӣм•„м„ң лӘЁлҚё лӮҙл¶Җм—җм„ң мқҙн•ҙн•  мҲҳ мһҲлҠ” лІЎн„° мһ„лІ л”©мңјлЎң ліҖнҷҳн•ҳлҠ” м—ӯн• мқ„ н•ңлӢӨ. лІЎн„°мқҳ м°Ёмӣҗ(кёёмқҙ)мқҖ лӘЁлҚёл§ҲлӢӨ лӢӨлҘҙлӮҳ, ліҙнҶө 768мқҙлӮҳ 1024 м •лҸ„лӢӨ.
CLIPм—җм„ң л§Ңл“Өм–ҙм§„ лІЎн„°лҠ” UNetмңјлЎң м „лӢ¬лҸјм„ң мқҙлҜём§Җ мғқм„ұмқ„ мң„н•ң мһ…л Ҙк°’мңјлЎң мӮ¬мҡ©лҗңлӢӨ.

UNetмқҖ CLIPмқҙ л§Ңл“  лІЎн„° к°’мқ„ л°ӣм•„м„ң мқҙлҜём§Җ мғқм„ұмқ„ мӢңмһ‘н•ңлӢӨ.
лӘЁлҚёлӢөкІҢ лҸҷмһ‘ л°©мӢқ мһҗмІҙлҠ” м •нҳ•м Ғмқҙм§Җ м•ҠлӢӨ. көімқҙ м„ӨлӘ…мқ„ н•ҳмһҗл©ҙ, "лһңлҚӨ л…ёмқҙмҰҲ"лҘј мөңмҙҲ н•ңлІҲ мҙҲкё°нҷ”н•ң лӢӨмқҢм—җ, кі„мҶҚн•ҙм„ң лЈЁн”„лҘј лҸҢл©ҙм„ң л…ёмқҙмҰҲлҘј мӨ„мқёлӢӨ.
мқҙ л…ёмқҙмҰҲлҘј мӨ„мқҙлҠ” кІғ мһҗмІҙк°Җ мӮ¬м „ н•ҷмҠөлҗң нҢЁн„ҙм—җ л”°лқјм„ң лІЎн„° мһ…л Ҙмқ„ кё°л°ҳмңјлЎң мҲҳн–үлҗҳлҠ” кІғмқҙкі , л…ёмқҙмҰҲк°Җ м¶©л¶„нһҲ мӨ„м–ҙл“Өм—ҲлӢӨкі  нҢҗлӢЁн•ҳл©ҙ л…ёмқҙмҰҲк°Җ мөңмҶҢнҷ”лҗң кІ°кіјл¬јмқ„ лӢӨмқҢ мҠӨн…қмқё VAEм—җ л„ҳкё°лҠ” кІғмқҙлӢӨ.

мӮ¬лһҢмқҙ к·ёлҰ¬лҠ” кІғмІҳлҹј лҸ„нҷ”м§Җм—җм„ң к·ёл ӨлӮҳк°ҖлҠ”кІҢ м•„лӢҲлқј, лҚ”лҹ¬мҡҙ лӮҷм„ңнҢҗм—җ м§Җмҡ°к°ңмҷҖ м—°н•„мқ„ мҚЁм„ң к·ёлҰјмқ„ мҷ„м„ұн•ҳлҠ” м…ҲмқҙлӢӨ.

VAE(Variational Auto Encoder)лҠ” UNetмқҳ кІ°кіјл¬јмқ„ л°ӣм•„м„ң м§„м§ң мқҙлҜём§Җ(PNG) л“ұмңјлЎң ліҖнҷҳн•ҳлҠ” м—ӯн• мқ„ н•ңлӢӨ.
кө¬мІҙм ҒмңјлЎң мҳҲлҘј л“Өл©ҙ, UNetмқҙ 64Г—64Г—4 нҒ¬кё°мқҳ м••м¶•лҗң latent н‘ңнҳ„мқ„ VAEм—җ лҚҳм§Җл©ҙ, VAE DecoderлҠ” к·ёкұё 512Г—512Г—3 нҒ¬кё°мқҳ мӢӨм ң н”Ҫм…Җ мқҙлҜём§ҖлЎң ліҖнҷҳн•ңлӢӨ.
мӢӨм ңлЎң л§Ңл“Өм–ҙм§ҖлҠ” мқҙлҜём§Җ нҒ¬кё°м—җ л№„н•ҙм„ң UNetм—җм„ң м—°мӮ°н•ҳлҠ” мқҙлҜём§Җмқҳ нҒ¬кё°лҠ” л§Өмҡ° мһ‘мқҖ нҺёмқҙкі , к·ёлһҳм„ң м•Ҫк°„мқҳ мҶҗмӢӨмқҙ мһҲмқ„ мҲҳлҠ” мһҲм§Җл§Ң мҶҚлҸ„лҘј л№ лҘҙкІҢ лҒҢм–ҙмҳ¬лҰҙ мҲҳ мһҲлҠ” кө¬мЎ°лҘј м–»мқ„ мҲҳ мһҲм—ҲлӢӨ.




лЎңм»¬м—җм„ң мӮ¬мҡ©н•ҙліҙкё°

StableDiffusionмқҖ мҳӨн”Ҳ лӘЁлҚёмқҙкё° л•Ңл¬ём—җ, к·ёлғҘ мһҗмң лЎӯкІҢ лҲ„кө¬лӮҳ лӢӨмҡҙл°ӣм•„м„ң мӮ¬мҡ©н•  мҲҳ мһҲлӢӨ.
GPUк°Җ м—Ҷм–ҙлҸ„ мӢӨн–үмқҖ н•  мҲҳ мһҲм§Җл§Ң лӘҮл¶„ м •лҸ„лЎң лҠҗлҰ¬лӢӨ.

uvмҷҖ pythonмқ„ мҚЁм„ң к°„лӢЁн•ҳкІҢ мӢӨн–үмқ„ н•ҙліҙкІ лӢӨ.
н•„мҡ”н•ң мў…мҶҚм„ұмқҖ лӢӨмқҢкіј к°ҷлӢӨ. uv pyprojectлӢӨ.

мҪ”л“ңлҠ” лӢӨмқҢкіј к°ҷлӢӨ.

from diffusers import StableDiffusionPipeline
import torch
from datetime import datetime

<br>

def main():
    print("=== Stable Diffusion CPU н…ҢмҠӨнҠё ===\n")

    print("1. лӘЁлҚё лЎңл”© мӨ‘... (мІ« мӢӨн–ү мӢң 4-5GB лӢӨмҡҙлЎңл“ңлҗ©лӢҲлӢӨ)")
    print("   кІҪлҹү лӘЁлҚё мӮ¬мҡ©: runwayml/stable-diffusion-v1-5")

    # CPUмҡ© лӘЁлҚё лЎңл“ң
    pipe = StableDiffusionPipeline.from_pretrained(
        "runwayml/stable-diffusion-v1-5",
        torch_dtype=torch.float32,  # CPUлҠ” float32 мӮ¬мҡ©
        safety_checker=None,  # мҶҚлҸ„ н–ҘмғҒмқ„ мң„н•ҙ safety checker л№„нҷңм„ұнҷ”
        use_auth_token=False,  # мқёмҰқ м—Ҷмқҙ кіөк°ң лӘЁлҚё мӮ¬мҡ©
    )
    pipe = pipe.to("cpu")

    # л©”лӘЁлҰ¬ мөңм Ғнҷ”
    pipe.enable_attention_slicing()

    print("вң“ лӘЁлҚё лЎңл”© мҷ„лЈҢ!\n")

    # н”„лЎ¬н”„нҠё
    prompt = "a serene mountain landscape at sunset, beautiful colors, digital art, highly detailed"

    print(f"2. мқҙлҜём§Җ мғқм„ұ мӨ‘...")
    print(f"   н”„лЎ¬н”„нҠё: {prompt}")
    print(f"   н•ҙмғҒлҸ„: 512x512")

    start_time = datetime.now()

    # мқҙлҜём§Җ мғқм„ұ
    image = pipe(
        prompt,
        height=512,
        width=512,
        num_inference_steps=20,  # мҶҚлҸ„лҘј мң„н•ҙ мӨ„мһ„
        guidance_scale=7.5,
    ).images[0]

    end_time = datetime.now()
    elapsed = (end_time - start_time).total_seconds()

    # м ҖмһҘ
    output_filename = f"output_{datetime.now().strftime('%Y%m%d_%H%M%S')}.png"
    image.save(output_filename)

    print(f"вң“ мғқм„ұ мҷ„лЈҢ!")
    print(f"   мҶҢмҡ” мӢңк°„: {elapsed:.1f}мҙҲ ({elapsed / 60:.1f}л¶„)")
    print(f"   м ҖмһҘ мң„м№ҳ: {output_filename}")

<br>

if __name__ == "__main__":
    main()

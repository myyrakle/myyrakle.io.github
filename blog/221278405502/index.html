[C++] main 반환타입 꾸미기

  • 2018.05.18
  • ::
  • 1 min read
#C++
[원본 링크]

c++은 c와는 다르게 반환타입을 강제로 int로 고정해놨다.

비주얼 스튜디오의 경우엔 void를 허용하긴 하는데 그건 그냥 비주얼이 해주는거고 딴놈들은 안해준다.

int main(void)... 너무 단조롭지 않은가.

좀더 멋있게 꾸밀수 있는 몇가지 방법을 소개한다.



auto 키워드를 쓴다. 그대신 ->int로 표기를 해줘야한다.

int main보다는 세련된 느낌이 든다.



decltype을 씌운것뿐인데 좀더 있어보이는 느낌이 든다.
하지만 이것도 -> 생략하면 추론 못한다.



전역변수를 넣어서 추론시켜봤다. main 내부의 값을 decltype에 넣으면 접근을 못한다...



개인적으로 이게 제일 맘에 든다.

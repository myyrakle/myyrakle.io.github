alpine에서 jemalloc 빌드가 깨지는 문제

  • 2025.04.02
  • ::
  • 1 min read
#Container
[원본 링크]

Docker 빌드시에 alpine 기반으로 jemalloc를 사용하려고 하면

높은 확률로 컴파일이 뻑날 수 있다.
이건 alpine이 기본으로 musl 기반 툴체인을 사용하는데, 그게 jemalloc의 기본 지원 대상에 속하지 않기 때문이다.

가장 일반적인 해결책은 그냥 jemalloc을 소스코드부터 받아서 빌드하는 방법이다.

RUN wget -O - https://github.com/jemalloc/jemalloc/releases/download/5.2.1/jemalloc-5.2.1.tar.bz2 | tar -xj && \
    cd jemalloc-5.2.1 && \
    ./configure && \
    make && \
    make install

그럼 좀 많이 느려지지만 돌긴 한다.
이미지를 미리 말아놓고 쓰는 것을 권장한다.

[Linux] 0% [Waiting for headers]

  • 2025.05.29
  • ::
  • 1 min read
#Linux
[원본 링크]

설치나 update를 하다보면, 뭔가 무한히 pending되는 것처럼 보일 때도 있다.

이건 그냥 네트워크가 느리거나, 원본 저장소에 문제가 있을 수도 있지만, 한번 apt-get 캐시를 날리고 reload하는 것만으로도 좀 나은 상황을 만들 수 있다.

sudo apt-get clean
cd /var/lib/apt
sudo mv lists lists.old
sudo mkdir -p lists/partial
sudo apt-get clean
sudo apt-get update

내 경우에는 꽤 유효했다.

[Rust] thread park

  • 2024.02.07
  • ::
  • 1 min read
#Rust
[원본 링크]

park는 Rust에서 제공하는 스레드 제어 기법 중 하나다.

이름대로 스레드를 주차한다는 뜻인데, 스레드에서 park를 사용하면 해당 스레드는 바로 정지한다. 이 상태에서는 CPU 리소스를 전혀 소모하지 않는다.

아래는 그에 대한 예제코드다.

use std::thread;

fn main() {
    let mut i = 0;
    let handle = thread::spawn(move || loop {
        println!("park! {i}");
        i += 1;
        thread::park(); // 로그 찍고 바로 주차합니다.
    });

    // 3초마다 unpark로 주차를 해제합니다.
    let controller = thread::spawn(move || loop {
        thread::sleep(std::time::Duration::from_secs(3));
        handle.thread().unpark();
    });

    controller.join().unwrap();
}

첫번째 스레드는 park를 통해 스스로를 중지시킨다.
두번째 스레드는 3초를 간격으로 다시 unpark를 풀어준다. 그럼 이때부터 중지했던 첫번째 스레드가 재시작한다.

이런 느낌으로 쓰면 된다.

[AWS] 코그니토: 사용자 풀 생성

  • 2020.09.21
  • ::
  • 2 min read
#AWS
[원본 링크]

코그니토는 AWS에서 자체적으로 제공하는 인증 및 권한 관리 서비스다.
간단히 말해서, 로그인 기능을 만들어서 제공해주는 것이다.

간단히 Node 람다 서비스에 붙여서 사용해보도록 하겠다.

먼저 코그니토 콘솔을 찾아 들어간다.

사용자 풀 관리 클릭

사용자 풀 생성

이름 설정 후 순서대로 진행

난 일단 이메일 주소로 로그인을 하도록 했다.

그리고 사용자의 속성을 지정한다.
주소, 닉네임, 생년월일, 폰번호, 이메일, 프로필사진, 성, ... 등등 웬만한 것들은 다 있다.

그리고 암호를 설정한다. 난 쓸데없는 특수문자와 대문자를 빼버렸다.
저런거 넣는다고 보안 수준이 엄청 늘어나는 것도 아니고.. 굳이 필요하진 않은 것 같다.

이건 뭔지 모르겠다. 넘긴다.

이것도 패스

패스

디바이스 기억 기능도 있다.
사용자가 접속한 기기, 위치, 운영체제를 기록해주는 기능이다.
나는 일단 사용하지 않았다.

이건 되게 중요한 부분인데, 생성 후에도 얼마든지 추가할 수 있으므로 일단 넘기겠다.

이건 가입이나 접속시 람다를 호출해서 이런저런 부가처리들을 가능케 해주는 부분이다.
필요한 기능이지만, 일단 넘어가겠다.

그리고.. 생성한다.

그럼 풀이 생성된다.

이제 이 풀을 통해서 사용자를 등록하고, 접속할 수 있다.

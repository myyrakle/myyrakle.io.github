  • ::

#AWS
[원본 링크]

AWS(Amazon Web Service)는 말 그대로, 아마존에서 제공하는 웹 호스팅 서비스다.
듣기로는 블랙 프라이데이 등의 호황 시즌을 감당하기 위해 서버를 대규모로 구축했다가, 평시에 남아도는 트래픽을 팔아먹으려고 만든 거였다고 한다.

2006년에 시작해, 클라우스 호스팅 서비스 중 굳건한 1위를 차지하고 있다. 구글이나 MS 등에게 추격을 받고는 있지만 여전히 가장 규모가 크고 대중적이다.

사이트에 들어가면 뭐 이것저것 많은데

기본적으로 가장 많이 사용되는 것이 EC2라는 녀석이다. Elastic Compute Cloud의 준말이란다.
그냥 가상 머신을 생성해서 그걸 서버로 돌아가게 해준다.

외에도 이것저것 참 많은데, 나중에 다시 다뤄보도록 하겠다. 나도 잘 모른다.

[Zig] 제어문

  • 2022.05.13
  • ::
  • 1 min read
#Zig
[원본 링크]

제어문은 if, switch, for, while 등이 제공된다.


if-else문

가장 일반적인 형태의 분기문이다.
if (조건식) {}
else if (조건식) {}
...
else {}
의 형태를 지닌다.

다음과 같이 사용할 수 있다.



if-else문 (표현식)

if-else문은 C의 삼항연산자처럼 표현식을 반환하는 형태로도 사용할 수 있다.
사용법은 어렵지 않다. 중괄호만 떼면 된다.



switch문

zig에서의 switch문은 패턴매칭이다.
rust의 것을 거의 그대로 베껴온듯하다.

다음과 같은 신택스를 가진다.
switch** (분기값) {**
**    패턴1 => ...,**
**    패턴2 => ...,**
**    else => ...,**
}

switch도 if-else문과 마찬가지로 표현식반환을 할 수 있다.

switch는 기본적으로 다음과 같은 형태로 사용할 수 있다.
리터럴값과 매칭되는 분기를 실행하는 것이다.

다만 좀 특별한 점은, 범위값 같은 패턴 표현을 사용할 수 있다는 것이다.



while문

우리가 아는 그 반복문 맞다.

신택스는 이렇다.
while** (반복조건식)**
{ ... }

다음과 같이 사용할 수 있다.



for문

for는 범위기반 반복문이다.
배열 등의 순회 가능한 값을 받아, 순서대로 순회를 한다.

신택스는 다음과 같다.
for** (배열값) |반복변수| { ... }**

다음과 같이 사용할 수 있다.

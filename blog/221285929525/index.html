#Rust
[원본 링크]

rust는 안타깝게도 아직 완성형으로 주어지는 IDE가 없다.

그냥 에디터에다 플러그인을 꼽아서 사용해야 하는데 필자가 vs code를 사용하고 있으므로 이걸 기준으로 쓰겠다.


자바를 쓰려면 jdk를 깔듯이 rust 컴파일러와 기타 패키지를 먼저 깔아줘야한다.

https://www.rust-lang.org/ko-KR/install.html

깔고, 설치파일을 실행하자.


그럼 시커먼 커맨드창이 뜨는데,

엔터 치고 디폴트로 그냥 깔면 된다.

설치가 끝나면

커맨드에 rustc나 cargo, rustup을 쳐서 이렇게 뭐가 빼곡하게 뜨면 성공이다.

Path는 알아서 깔아준다.


이제 에디터인 vs code를 깔아보자.

https://code.visualstudio.com

깔고

켠다.

그냥 yes맨이 되어도 좋고, 이렇게 입맛대로 골라도 된다.


켠다.

뭐라뭐라 잔소리하는데 코리언 랭귀지만 깔아주자.

Git은 당장 필요한건 아니니까 무시하면 된다.


확장에 들어가서 rust를 검색한다.

그리고 위에서 Rust(rls)를 깔아준다.



이제 rust 파일임을 인식하게 해주는.. rust 프로젝트를 만들어준다.

커맨드를 켜자.


요래 치면


사용자 디렉토리에 폴더가 생긴다.


이제 vs code에서,


작업 영역에 폴더 추가나 저기 폴더 열기를 눌러서


아까 그 폴더를 연다.


그럼 안에 main 소스파일과 기타등등이 들어있다.

왜인진 모르겠는데 껐다키면 저렇게 아래에 뭐가 뜨는데,
저기서 legacy를 눌러주자.


작업실행 누르고

cargo run을 눌러주자.

build하면 빌드는 되는데 실행이 안된다.


그럼 대강 된다.

[Clojure] 벡터

  • 2019.12.01
  • ::
  • 2 min read
#Lisp/Clojure
[원본 링크]

벡터는 클로저에서 제공하는 내장 배열 타입이다.
대괄호로 감싼 시퀀스로 표현된다.


근데 대부분의 함수형 언어들이 그렇듯이, 클로저도 타입검사같은걸 제대로 안한다.
그래서 정수 벡터에 문자열같은걸 끼워놔도 잘 돈다.

이런게 꼴보기 싫다면 생성용 함수를 쓰면 된다.


vector-of
벡터 생성용 함수다. 첫번째 인자로 타입값을 받고, 타입검사를 해준다는 게 특이사항이다.
(vector-of 타입 인자1 인자2...)



nth함수
인덱스를 받고 해당 요소를 반환한다.
(nth 벡터 인덱스)

근데 접근범위를 벗어나면 예외를 던진다.

이런 예외상황을 막으려면, 인자를 하나 더 줘서 예외발생시 반환 표현식을 정해줄 수 있다.
(nth 벡터 인덱스 예외표현식)


get함수
이것도 인덱스를 받고 해당 값을 반환한다.
nth와 뭔 차인지 모르겠다.
(get 벡터 인덱스)
(get 벡터 인덱스 예외표현식)



conj 함수
벡터 뒤에 요소를 추가한다.
리스트와 다르게 벡터는 뒤에서 삽입이나 삭제를 수행한다.
(conj 벡터 값1 값2...)



pop함수
벡터 뒤의 인자를 하나 제거한다.
(pop 벡터)

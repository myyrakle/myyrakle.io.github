#Rust
[원본 링크]

cargo-asm은 rust로 빌드된 코드를 어셈블리 수준에서 리버싱할 수 있게 도와주는 편리한 보조도구다.
임베디드 같은 마이크로한 최적화가 필요한 환경에서 사용할만하다.

설치는 간단하다.

cargo install cargo-asm

사용법도 어렵지는 않다.

프로젝트 루트 경로에서 cargo asm을 실행하면, 전역에 위치한 심볼들을 조회할 수 있다.

저기서 각 함수의 내용을 보고 싶다면, 그걸 전달해서 다시 실행하면 된다.

쭉 나온다.

비슷한 방식으로 llvm-ir도 뽑아서 볼 수 있다.

그렇다.

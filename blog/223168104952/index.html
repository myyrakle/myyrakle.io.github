[Github] Copilot Chat (Beta)

  • 2023.07.28
  • ::
  • 1 min read
#Github
[원본 링크]

Copliot Chat은 최근에 추가된 개발자용 AI 채팅봇이다.
현재 베타테스트고, 코파일럿 가입자만 사용 가능하다. (1년 100달러)




테스트 참여하기

아래 링크에 들어가서 대기열에 집어넣는다.
https://github.com/github-copilot/chat_waitlist_signup/join

그래서 이렇게 되면 사용이 가능해진 것이다.




Vscode 세팅

현재는 VSCode로만 지원하는 것 같다.
확장을 하나 깔아주면 된다.

그리고 로그인도 되고 다 된다면 이렇게 뜰텐데

여기다가 물어보고 싶은 것을 마음대로 물어볼 수 있다.

그냥 답변에 대한 퀄리티는 조금 더 써봐야 알 것 같고, 성능은 아주 빠르고 좋았다.
이제 ChatGPT나 Bing AI는 굳이 쓰지 않을 것 같다.




장점: 코드 선택

이 확장 도구의 강력한 장점 중 하나는, VSCode 에디터와 연동해서 상호작용이 가능하단 것이다.
노가다를 줄이기에 용이하다.

예를 들어, 이런 데이터 타입이 있고, 타입에 대한 빌더 코드를 만들고 싶다면

우선 그 코드 영역을 선택한 다음

그냥 채팅으로 물어보면 된다.

이거 말고도 코드 평가(리뷰)나 단위테스트 생성 등 잡다한 코드 노가다를 짬때릴 수 있다.

코드리뷰는 언어의 특성을 파악하지 못하거나 애매한 부분이 조금 보이긴 하는데, 그래도 그런대로 쓸만은 했다.

일단 좀더 써봐야 알 것 같다.

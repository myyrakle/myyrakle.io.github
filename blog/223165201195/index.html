cobra는 대표적인 Go의 cli 제작용 도구다.



설치 (CLI)

이건 생으로 다 짜려면 번거롭고 불편하게 생겨먹어서, 제공되는 cli를 써야 좀 쓸만하다.
먼저 설치해준다.

go install github.com/spf13/cobra-cli@latest

그래서 이렇게 실행이 되면 잘 깔린 것이다.

이것도 cli를 써서 추가해주면 된다.

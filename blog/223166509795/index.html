mockery는 mock 자동화 보조도구다.
gomock과 라이벌인거같다.




설치

먼저 cli를 설치해줘야 한다.

go install github.com/vektra/mockery/v2@v2.32.0

이렇게 실행이 되면 잘 깔린 것이다.

Mock 살펴보기

Mock 생성물이라 해봐야 사실 별거 없다.
원본 인터페이스의 각 메서드에 대해서 empty 구현을 두고, 테스트 시점에 주입할 수 있도록 하는 것이다.

그래서 이런 의미없는 보일러플레이트들만 잔뜩 있다.

이걸 직접 분석하거나 관리할 일은 없다. mockery cli로 싱크만 계속 맞춰주면 된다.




테스트 작성하기

그럼 저 mock 생성물을 이용해서 간단한 단위테스트를 작성해보겠다.
mock 객체를 생성하고, On으로 각 메서드가 어떤 조건으로 호출되었을때 어떤 값이 리턴되도록 할지를 정의하면 된다.

그 이후에는 그냥 go test 내장기능을 이용해서 테스트를 수행하면 된다.

[Rust] 트레잇

  • 2018.06.02
  • ::
  • 1 min read
#Rust
[원본 링크]

Rust에서 Trait은 타 언어의 인터페이스에 해당하는 개념이다.

선언법은

trait 트레잇이름{
메소드 선언부;
메소드 선언부;
...
}
요래 쓰면 된다.

그리고 트레잇을 타입에 구현할 경우

impl 구현할트레잇 for 구현대상타입
{
선언부만 있던 메소드 구현
...
}
이렇게 하면 된다.

이것도 별거 없다. 그대신 저런식의 트레잇 구현부에는 해당 트레잇 메소드의 구현만 가능하고, 트레잇에 없는 건 못써넣는다.

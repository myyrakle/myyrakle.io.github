swaggo는 swagger 생성용 모듈이다.
comment 기반으로 generate를 해주는 도구다.




설치

먼저 CLI 도구를 설치한다.

go get -d github.com/swaggo/swag/cmd/swag

그래서 이렇게 실행이 되면 된다.

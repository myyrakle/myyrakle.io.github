#라이브러리
[원본 링크]

https://github.com/myyrakle/gopring/releases/tag/v0.1.4
어노테이션 떡칠로 자바스럽게 쓸 수 있도록 구성을 해봤다.
최종 결과물은 echo + uber fx의 래퍼다.

아직 부실하고 없는게 많은데, 거의 겉모양만 대충 만들어놔서 그렇다. 짬짬이로 조금씩 채워나가보려 한다.




CLI 설치

전용 cli 도구를 먼저 설치해야 한다.




프로젝트 생성

new 명령으로 프로젝트를 생성한다.

그럼 이런식으로 프로젝트가 생성된다.

프로젝트 구조 규칙은, 코드는 전부 src에 있어야 한다는 것이다. 그것만 지키면 된다.




실행

run 명령을 쓰면 자동으로 컴파일을 해준다.

그럼 dist라는 출력 경로에 결과물을 만든 다음에 실행을 한다.


현재는 echo 기반이다.


이런식으로 실행은 대충 된다.




빌드

실행파일을 뱉으려면 build를 날리면 된다.




문법

자바의 멋진 신택스를 본받아 주석 기반의 어노테이션을 구현했다.
서비스는 이렇게 만들고, 자동으로 의존성 주입이 된다.

route는 컨트롤러와 매핑 어노테이션으로 정의된다.

그렇다.

**반환 없는 함수**  
일반적으로 자바와 비슷한 방식으로 선언하면 반환하지 않는(사실 Unit을 반환하는) 함수가 된다.  
권장되진 않는다.  

**def** 함수명(인자...)  
{ ... }  

인자는 아래와 같이 표현한다.  
**인자명:인자타입**  

아래 코드는 정수를 하나 받아 출력하는 반환 없는 함수 put을 정의한 것이다.
![image](/images/img_9f91a3123635ee77.gif)

![image](/images/img_9f91a3123635ee77.gif)
그런데 이러한 방식은 권장되지 않는 선언법이다. 반환을 할수가 없기 때문이다.  

<br>

무조건 =를 붙여줘야 한다.   

**def** 함수명(인자들...): 반환타입 =  
{ ... }

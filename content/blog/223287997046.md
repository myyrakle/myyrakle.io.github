보통 별다른 설정을 하지 않으면 nano로 에디터가 구동되는 경우가 많은 것 같다.  

![](/images/img_682001b39f5fa0d4.png)
나는 나노가 별로다.  

vi로 에디터를 변경하고 싶으면 이렇게 하면 되고  

```javascript
git config --global core.editor "vi"
```

![](/images/img_d07702f4aaa12f08.png)

![](/images/img_edaee64c5e3edce7.png)

vscode로 쓰고 싶다면 이렇게 하면 된다.  

```javascript
git config --global core.editor "code --wait"
```

![](/images/img_50ced26e2f8c2d74.png)

![](/images/img_420f63ca0405da93.png)

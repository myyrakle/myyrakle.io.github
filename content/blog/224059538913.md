Claude Code는 Claude에서 제공하는 CLI 기반의 개발 보조용 에이전트 도구다.  

CLI 기반 에이전트 열풍을 불러온 주범이기도 하다. 지금 시점에서는 이 방면에서 거의 독보적인 위치를 차지하고 있다.  

<br>

<br>

<br>

## 비용

Claude는 무료 사용량을 약간 제공하긴 하지만, 매우 제한적이다.  
API를 직접 쏘거나 Claude Code를 써야 한다면 17달러부터 시작하는 구독제나, 종량제를 써야 한다.  
얘들은 Trial도 제공하지 않기 때문에 찍먹이라도 해보려면 결제를 해야 한다.  

![](/images/img_cd7b62ebd8517577.png)

![](/images/img_4a30b5dba98f8ae8.png)
난 17달러 구독제로 질러봤다.  

비싼 비용과 토큰 소모량 때문에 망설이다가, 최근 Copilot의 품질 저하를 많이 느껴서 한번 경험해보려고 끊었다.

## CLI 설치

CLI은 npm을 통해 설치할 수 있다. 왜 굳이 npm인지는 잘 모르겠지만...  
권한설정도 어딘가 이상해서 sudo가 필요할 수도 있다.  

```javascript
sudo npm install -g @anthropic-ai/claude-code
```

![](/images/img_cff00f389e239ca8.png)

설치가 되면, claude라는 실행파일로 바로 실행해볼 수 있다.  

![](/images/img_a71e5ab88b866daf.png)
그러면 이렇게 대화형 세션이 열리면서, 사용할 수 있게 된다.  

<br>

![](/images/img_bbb6520c04f667e9.png)
돈을 내지 않았다면 컷당한다.  

<br>

## VSCode에서 사용하기

VSCode에서도 확장을 직접 제공한다.  

![](/images/img_cd376017f258f41d.png)

![](/images/img_713792e65014a2f8.png)
마켓플레이스에서 직접 설치해서 사용할 수 있다.  
VSCode 확장이 기존 CLI 도구와 다른 점은. AWS Bedrock 같은 서드파티 소스도 사용할 수 있다는 것이다.  

<br>

![](/images/img_19681d8fbd77cb0c.png)

![](/images/img_45f093fec6d80705.png)
설치하면 우상단에 버튼이 생기고, 채팅창을 열 수 있게 된다.  
CLI와 컨텍스트를 공유하고, 사용법도 거의 동일하다.  

근데 이거 ux가 좀 이상하다. 다른 확장들은 왼쪽 확장 전용 패널에 예쁘게 모아둘 수 있는데, 이건 그게 안된다.  
일반적인 "코드 창"과 동일한 레벨로 관리가 되더라.  

<br>

![](/images/img_6ef9d237123a9e7f.png)

다른 에이전트 도구들에 비해서 사고 방식이나 해결 방식이 좀 더 체계적인 부분이 많이 느껴진다.

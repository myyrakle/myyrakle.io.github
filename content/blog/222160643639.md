nginx는 apache에 대항하며 나온 서버용 '리버스 프록시' 도구다.  
실제 서버와 클라이언트 사이를 중개하면서 잡다한 처리들을 대행해주는 녀석이라 하겠다.  

centos7에서는 다음과 같이 설치할 수 있다.  
기본 패키지 목록에 없어서 손을 좀 써야한다.  
먼저 파일을 만들고  

```javascript
sudo vi /etc/yum.repos.d/nginx.repo
```

![](/images/img_0734c43f36bae1bb.png)

다음과 같이 작성한다.  

```javascript
[nginx]
name=nginx repo
baseurl=http://nginx.org/packages/centos/7/$basearch/
gpgcheck=0
enabled=1
```
여기서 링크가 [http://nginx.org/packages/centos/7/$basearch/](http://nginx.org/packages/centos/7/%24basearch%2F)로 되어있는데.  
다른 os 버전은 centos 자리에 그 이름을 넣고 7 자리에 버전을 넣어서 바꾸면 된다.  

<br>

![](/images/img_9d90063b51f8d0fe.png)

![](/images/img_1e8335d6b02f08a8.png)

그리고 방화벽을 열어준다.  

```javascript
sudo firewall-cmd --permanent --zone=public --add-service=http
sudo firewall-cmd --permanent --zone=public --add-service=https
sudo firewall-cmd --reload
```

![](/images/img_5f75caa28fb27060.png)

그리고 systemctl로 시작, 자동실행이 가능하도록 해주면 끝이다.  

```javascript
systemctl start nginx
systemctl enable nginx
```

![](/images/img_d40d24ae19d88d32.png)

그리고 루프백으로 접근해서 이렇게 뜨면 된다.  
다른 설정을 하지 않았을 경우에 기본포트는 80이다.  

![](/images/img_d07262d507ff7187.png)

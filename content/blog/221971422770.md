nodemon은 node를 자동으로 재실행시켜주고 관리해주는 편리한 보조 도구다.  

이런건 전역으로 깔아주는 게 좋은 것 같다.  

![](/images/img_d32637bef8b34b49.png)

그리고 대강 프로젝트를 파고, 소스를 하나 짜둔다.  

![](/images/img_b3d680682f2c906c.png)

그럼 이제 실행할 때 node 대신 nodemon을 사용하면 되는 것이다.  

![](/images/img_de923e93f9c55610.png)
그럼 소스파일이 수정될때마다 자동으로 재실행될 것이다.  

이외에도, package.json을 수정해서 nodemon 관련 설정을 관리할 수도 있다.  
아래와 같은 포맷을 nodemonConfig로 넣어주면 된다.  

```javascript
"nodemonConfig": {
    "name": "nodemon",
    "homepage": "http://nodemon.io",
    "...": "... other standard package.json values",
    "nodemonConfig": {
      "ignore": ["test/*", "docs/*"],
      "delay": "2500"
    }
}
```

![](/images/img_0bc829e48eb3426b.png)
ignore는 재실행 반영 대상에서 제외시키는 것 같다.

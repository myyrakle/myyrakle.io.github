LINQ는 언어에 통합된 쿼리(**Language Integrated Query**)의 준말로,   
언어 구문 내에서 SQL 스타일의 쿼리를 통해 복합 데이터를 처리하게 해주는 꽤 멋지고 편한 녀석이다.  

한번 천천히 살펴보자  

**from ... in ... select ... 구문**  
linq에 있어서 가장 기본이 되는 것이 바로 from 절과 select 절이다.  
from은 무슨 데이터를 어떻게 가져올지를 표현한다.  
그리고 select는 현재 선택된 데이터를 어떻게 반환할 지를 표현한다.  

한번 linq 구문을 사용해서 배열 데이터를 건드리지 않고 그대로, 배출해보겠다.

**걸러내기: where 절**  
위에서는 from 절에서 가져온 모든 요소를 반환했었다. 하지만 필요한 요소를 선택적으로 받아야 할 경우도 많다.  
where절은 그럴 때 사용한다.  

where 절의 표현식에는 조건식이 들어가야 한다. 그리고 해당 조건식에 부합하는 요소들만 남기고 전부 버린다.  

만약 정수 배열에서 홀수만 남기고 버리고 싶다면, 이렇게 하면 된다.

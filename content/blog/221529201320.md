타입스크립트 컴파일러를 먼저 깔아야 하는데,  
가장 편한 방법으론 2가지가 있다.  
1.비주얼 스튜디오로 깐다.  
2.npm으로 깐다.  

비주얼 스튜디오를 평소에 즐겨쓴다면 1번이 좋은 선택인 것 같다.  
나도 그렇게 했고.  

![](/images/img_20d53a90ce41e0ce.png)
이렇게... 멋지게 제공해주기 때문이다.  

비주얼스튜디오에서 깔면... 아무래도 파일을 막 숨겨놓다보니 찾아서 패스를 깔아줘야 할텐데, 대충 이런 곳에 있을것이다.  

```javascript
C:\Program Files (x86)\Microsoft SDKs\TypeScript\0.8.0.0
```

<br>

비주얼 스튜디오 안쓰고 npm에서만 깔고싶다면 이러면 된다. 간단하다.  

```javascript
npm install -g typescript
```

<br>

<br>

![](/images/img_62d83cb468a752b0.png)
tsc가 컴파일러 명이다.  
이렇게 잘 뜨면 된다.  

<br>

아 그리고 웹 컴파일러를 사용해도 좋다.  
단순한 학습용이라면 이것도 상당히 괜찮은 방법이라 생각된다.

Hack은 PHP를 마개조한 끔찍한 혼종이다.  
PHP의 태생적으로 구린 성능과 확장성을 최적화하기 위해서 자체적으로 불법개조시술을 가한 것이다.  
그래서 PHP와 호환되지는 않는 별개의 생명체다.  

페이스북과 슬랙 메인 백엔드에 쓰이는 것으로 유명하다.  
페북이 php 쓴다고 php 억빠하는 사람들이 많은데, 사실 쟤들이 쓰는건 hack이라서 php 쓴다고 하기도 애매하다.  

2011년에 나왔고, 페이스북에서 직접 만들고 관리한다.

## HHVM 설치

Hack은 HHVM이라고 해서 자체적인 가상머신 플랫폼 위에서 동작한다. HH는 놀랍게도 힙합의 축약이다.  
windows는 지원하지 않고, 리눅스와 Mac만 지원한다.  

내 테스트 환경은 우분투 22다.  
아래 명령을 실행한다.  

```javascript
sudo apt-get update &&
sudo apt-get install --yes software-properties-common apt-transport-https &&
sudo apt-key adv --recv-keys --keyserver hkp://keyserver.ubuntu.com:80 0xB4112585D386EB94 &&
sudo add-apt-repository 'deb https://dl.hhvm.com/universal release main' &&
sudo apt-get update &&
sudo apt-get install --yes hhvm &&
hhvm --version
```

## 프로젝트 구성하기

폴더 하나를 만들고, 그 안에 hhconfig를 추가한다.  

```javascript
mkdir my_project
cd my_project
touch .hhconfig
```

![](/images/img_fa56422e202f89a8.png)
hhconfig는 해당 디렉터리가 hack 프로젝트의 루트 디렉터리임을 알려주는 매니페스트 파일이다.  
nodejs의 package.json이나 go의 go.mod, rust의 cargo.toml과 비슷하다고 보면 된다.

## Hello World 프로그램

그리고 다음 소스코드를 .hack 확장자로 작성한다.  

```javascript
use namespace HH\Lib\IO;

<<__EntryPoint>>
async function main(): Awaitable<void> {
  await IO\request_output()->writeAllAsync("Hello World!\n");
}
```

![](/images/img_15ba4aaf47c0b3d2.png)
순수 php와 비교해서 꽤 이질적인 것을 볼 수 잇다.  
EntryPoint 어트리뷰트를 통해 진입점을 명시하고, 해당 진입점 함수에서 텍스트를 출력한다.  
저기서 출력하는 텍스트가 일반적으로 client에게 반환되는 응답값이 된다.

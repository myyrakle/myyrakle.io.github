터미널 환경에서도 git 로그인을 저장해서 추가 로그인 없이 액세스를 하고 싶을 수 있다.  
그럴 때는 다음 명령을 입력한 후  

<br>

```javascript
git config credential.helper store
```

git push, pull 등으로 로그인에 성공하면, 다음부터는 로그인이 필요하지 않다.

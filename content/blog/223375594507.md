[https://github.com/myyrakle/nosyman](https://github.com/myyrakle/nosyman)  
꽤나 열심히 사용하고 있는 온프레미스 빌드머신이 있는데, 필연적으로 발생하는 디스크 용량 초과 문제를 좀 조기에 발견해서 정리하기 위한 목적으로 만든 보조도구다.  

디스크 사용량이 임계치를 넘기면 슬랙으로 자동 알림을 보낸다. 검사 간격은 30초다.  

설치는 cargo를 통해서 하면 된다.  

```javascript
cargo install nosyman
sudo ln -s /home/$USER/.cargo/bin/nosyman /usr/bin/nosyman
```

![](/images/img_9ae10b893adaa363.png)
sudo 실행을 위한 심볼릭 링크만 추가로 달아준다.

## 사용법

먼저 설정파일을 초기화하고, 시스템을 systemd에 올려야 한다.  
아래 명령이 그걸 다 해준다.  

```javascript
sudo nosyman --init
```

![](/images/img_a568d5b9f1aaca99.png)
대화형으로 앱 토큰, 슬랙 채널 ID, 디스크 임계치를 입력한다.  

슬랙 앱 사용법은 아래 포스트를 참고한다.  
[https://blog.naver.com/sssang97/223176628812](https://blog.naver.com/sssang97/223176628812)  

저래놓으면 자동으로 프로세스가 뜬다.  

![](/images/img_dbacf801360963e6.png)

그리고 임계치를 넘어설때마다 이렇게 알림을 보내줄 것이다.  

![](/images/img_6fa2c18c3645933f.jpg)
Open-Closed 형식이라서 초과한 상태라고 계속 스팸을 보내진 않는다. 한번만 보낸다.  
내려갔다가 올라가면 다시 보낼 것이다.

작동이 확인된건 msvc, gcc, clang 뿐이다.  
다른 컴파일러에서는 동작하지 않는다.  

var은 변수형.  
let은 상수형이다.  
스위프트에서 착안했다.  

<br>

#ifdef __GNUC__  
    #define var __auto_type  
 #define let const __auto_type  
#elif defined __clang__  
    #define var __auto_type  
    #define let const __auto_type  
#elif defined _MSC_VER  
    #define var auto  
    #define let constauto  
#else  
 #define var no_var  
    #define let no_let  
#endif

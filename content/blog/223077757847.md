nats는 JetStream 확장을 통해 Key-Value 데이터베이스 시스템의 기능도 지원한다.  

JetStream 기반으로 동작하기 때문에 메모리가 아니라 disk 기반으로 저장된다.  

<br>

<br>

<br>

## 버킷 만들기

버킷은 테이블 단위라고 보면 된다.  
하나의 버킷에는 하나 이상의 키-값 쌍들이 들어갈 수 있다.  

버킷은 이렇게 만들 수 있다.  

```javascript
nats kv add 버킷명
```

![](/images/img_c6a6001ad606b796.png)

kv ls과 kv info 명령을 사용하면 이후에도 버킷의 정보를 조회할 수 있다.  

![](/images/img_52b929f28404e4a7.png)

![](/images/img_e793320d70bf46ee.png)

## 데이터 액세스

키값 쌍을 새로 할당하거나 수정할때는 put 명령,  
조회할 때는 get 명령을 사용하면 된다.  

```javascript
nats kv put 버킷 key value
nats kv get 버킷 key
```

![](/images/img_8d258c15e7983925.png)
별로 어려울 것은 없다.  

기본 형태는 이게 다다.

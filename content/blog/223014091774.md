mac에서는 다른 도구들이 대개 그렇듯, brew를 쓰는 편이 편리하다.  
다음과 같이 설치하고  

```javascript
brew install redis
```

![](/images/img_4319cb4842911915.png)

시작시키면 된다.  

```javascript
brew services restart redis
```

![](/images/img_5139dcba32a236d2.png)

그럼 cli로 접속이 될 것이다.  

![](/images/img_4773180a15a6e943.png)

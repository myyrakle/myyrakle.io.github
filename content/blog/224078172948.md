codewiki는 최근에 구글에서 낸 LLM 기반 프로젝트 문서화 플랫폼이다.  
Github 등에 게시된 오픈소스 프로젝트들에 LLM으로 분석을 돌려서 위키를 생성해주는 것이다.  

이건 문서화를 일반 사용자가 직접 제어할 수는 없고, 검색엔진처럼 그냥 구글에서 자기들이 만들어서 목록에 추가하는 식이다. 생성 요청은 가능하다.  

그리고 현재는 public repository에 대해서만 지원이 된다. 향후에는 private repository에 대해서도 기능 추가를 고려하고 있다고 한다.  

링크 참조  
[https://codewiki.google/](https://codewiki.google/)

이런걸 통해서 프로젝트에 대한 기여를 빠르게 할 수 있도록 보조해주는게 플랫폼의 목적이다.

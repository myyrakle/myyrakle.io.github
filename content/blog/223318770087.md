![](/images/img_9e1f1e175280a8d9.png)
vite는 node.js 환경의 주요 보조도구 중 하나다. Vue 개발자로 유명한 에반유가 만들었다.  
그래서 좀 찝찝하긴 한데... Vue 외에도 다 사용은 할 수 있는 범용 도구다.  

포지션이나 용도는 대충 웹팩과 비슷하다고 보면 된다. 번들링, dev 서버 띄우기 등의 기능을 제공한다.  
개발모드로 돌릴때 로드가 매우 빠르다는 장점이 있다.  

<br>

<br>

<br>

## 프로젝트 생성하기

npm create 명령을 통해 손쉽게 프로젝트를 구성할 수 있다.  

```javascript
npm create vite@latest
```

![](/images/img_db475a13f9aa12e5.png)
그러면 무슨 포맷으로 생성할지 물어본다. 대중적인건 대충 넣어놨다.  
나는 국밥 리액트로 골랐다.  

<br>

![](/images/img_75f2867394a0dfbc.png)
그러면 언어와 툴체인까지 골라서 생성을 할 수 있다.  

<br>

![](/images/img_7704e8b44f01bd47.png)

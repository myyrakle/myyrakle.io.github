GEMINI CLI는 구글에서 공개한 CLI 도구다.  

이것도 발표하면서 대단한 혁신인 것처럼 과대광고를 잔뜩 뿌려댔는데, 그건 아니고 본질적으로는 그냥 Gemini API를 CLI로 쏘면서 쓸 수 있게 해주는 도구에 불과하다.  

<br>

<br>

<br>

## 설치

설치는 또 npm 설치 권장이더라.  
적당히 깔면 된다.  

```javascript
npm install -g @google/gemini-clinpm install -g @google/gemini-cli
```

그러면 gemini라는 이름으로 실행할 수 있다.  

![](/images/img_fc748f06864d38d3.png)
이렇게 뜨면 잘 깔린 것이다.  

<br>

<br>

<br>

## 기본 사용법

처음 실행하면 대화형 인터페이스 띄우면서 옵션을 고르라고 한다.  

![](/images/img_8a5f5e904b30f8e8.png)
따로 키 없이도 구글계정만으로도 사용이 가능하다.  

내 경우에는 그냥 구글계정으로 연동했다.  

![](/images/img_a4c0f910e5323ccf.png)
그러면 브라우저 띄워서 계정 선택하라고 한다.  

<br>

![](/images/img_7ad9ddf29d2d19e1.png)
로그인해서 적당히 승인하도록 하면 된다.  
Gmail 로그인으로 연동했을 경우에는 분당 60회, 하루 1000회의 요청이 무료로 제공된다.  

그러면 이제 터미널 UI로 채팅을 치면서 써볼 수 있다.  

![](/images/img_d42aa5d17c558ece.png)

![](/images/img_49eb24ce1ab1949c.png)
터미널로 꽤 근사하게 잘 만들어두긴 했다.

vitest는 vite 기반으로 동작하는 단위테스트 도구다. jest와 사용 형태는 거의 동일하다.  
편리한 설정, 빠른 실행 성능을 장점으로 내세운다.  

<br>

<br>

<br>

<br>

## 세팅

먼저 디펜던시로 설치하고  

```javascript
npm install -D vitest
```

![](/images/img_89dd65aeac673bd6.png)

커맨드에 테스트 실행을 추가한다.  

![](/images/img_905504d681cd498d.png)

그래서 이렇게 실행이 되면 잘 된 것이다.  

![](/images/img_b6ff6825384246ed.png)
여기서 또 추가로 설정해야하는건 없다.  
jest와는 다르게 js나 ts나 바로 된다.  

<br>

<br>

<br>

## 기본 사용법

jest와 별로 다를건 없다.  

![](/images/img_a2224f3832e3f2ca.png)
이런 함수가 있다면  

<br>

![](/images/img_291d9459522162af.png)
파일명이 *.test.*인 파일을 추가, 테스트코드를 적당히 작성한다.  

<br>

![](/images/img_dc86b5bfa3054906.png)
그리고 커맨드를 돌리면 된다.  
성공했다고 한다.  

test가 실행중일때는 코드를 수정해서 저장하면 즉시 반영된다.  

![](/images/img_22ea4e8a7a19f136.png)
이런 식으로 말이다.

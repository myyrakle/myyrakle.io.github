brew로 설치하는게 편리하다.  
먼저 brew로 깔고

```javascript
brew install postgresql
```

![](/images/img_4b367ad74d033ecc.png)

데몬을 켜준다.

```javascript
brew services start postgresql
```

![](/images/img_05c38d128d074b37.png)

커밋을 정신없이 치다보면, 마지막으로 찍었던 커밋을 수정하거나, 마지막 커밋에 변경사항을 추가하고 싶을 때가 있다. 이럴때는 reset이나 rebase로 해결을 할 수도 있지만, 간편한 방법이 있다.  
바로 amend다.  

만약 다음과 같이 최신 커밋이 있고  

![](/images/img_efb07f6ad54ea8c2.png)
저 커밋을 수정하거나 덧대고 싶다면  

다음과 같이 명령을 날린다.  

```javascript
git add . # (필수는 아님)
git commit --amend
```

![](/images/img_74b6722a4ba51661.png)
변경사항을 더하고 싶다면 add로 올리고 해도 되는데, 변경사항 없이도 수행할 수 있다.  

여튼 그러면 편집창이 뜰 것이다.  

![](/images/img_1ddab535fc558b64.png)
여기서 커밋 메세지를 수정하거나 해도 되고  

파일을 저장 후 닫으면  

![](/images/img_6fac6aed7a9ffcd6.png)
해당 변경사항이 새로 커밋될 것이다.  

이건 기존 커밋을 지우고 새로 만든 버전의 커밋으로 대체하기 떄문에, 최신 변경사항이 이미 remote에 push된 상태라면 force push를 해야 한다.  
이렇게 말이다.  

![](/images/img_b0a4cf431a3d6353.png)

아무튼 그러면 잘 반영될 것이다.  

![](/images/img_0b91c18f045e00db.png)

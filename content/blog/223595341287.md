## 기본 사용법

설치는 cargo로 받아도 되고,  

```javascript
cargo install musica-app
musica-app
```

릴리즈 페이지에서 받아도 된다.  
[https://github.com/myyrakle/musica/releases/tag/v0.1.0](https://github.com/myyrakle/musica/releases/tag/v0.1.0)  
windows는 빌드를 안해봐서 모르겠는데.. 아마 되지 않을까?  
mac은 디렉터리를 다이얼로그 열어서 선택하는 것만 빼면 잘 돌 것이다. 이건 갖다쓴 native dialog 라이브러리에 버그가 있어서 고쳐야한다.  

UI는 최대한 단순하게 만들었다.

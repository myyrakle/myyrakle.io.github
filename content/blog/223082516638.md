![](/images/img_77122cb9f3280f7c.png)
Cloud Spanner는 GCP에서 자체적으로 구현, 구축해서 제공하는 초대형 분산 관계형 데이터베이스 서비스다.  
전세계에 지리적으로 분포되는 시스템에 적합하다.  

현존하는 멀티리전-멀티노드-관계형 데이터베이스를 제공하는 클라우드 서비스 중에서 가장 빼어난 성능과 완성도를 자랑한다.  

근래 부상한 CockroachDB만 해도 Spanner 출신 개발자들이 뛰쳐나와서 경쟁하려고 거의 비슷하게 만든 것인데, 아직 그 아성에 도달할 정도는 아니라는 것 같다.

## 비용

자세한 것은 사이트를 참조한다.  
[https://cloud.google.com/spanner/pricing?hl=ko](https://cloud.google.com/spanner/pricing?hl=ko)  

이만한 서비스가 딱히 없는만큼, 가격도 매우 비싼 편이다.  

DB 노드 개수, 멀티리전 여부, 사용한 스토리지 총량을 기준으로 요금을 부과한다.  
데이터 전송 같은건 또 그것대로 비용이 나갈 것이다  

일단 실제로 노드가 떠있는 개수와 시간에 비례해서 고정 요금이 나간다.  

![](/images/img_314053f9aa0e155e.png)
일단 서울에 그냥 노드를 하나만 띄운다고 해도, 시간당 1.17 달러니까 한달에 거의 5만원 돈이 나간다.  

<br>

![](/images/img_06b9c92c750584f9.png)
근데 또 멀티리전으로 하면 시간당 3.90달러, 한달에 117달러=15만원 정도 나간다.  

또 사용한 스토리지 용량만큼도 비용을 지불한다.  

![](/images/img_efd49f73c82b819b.png)

백업 스토리지는 기본 스토리지보다는 싸지만, 그래도 여전히 비싸다.  

![](/images/img_cf97d39316ea2cab.png)
이러면 아무리 작게잡아도 최소 한달에 백만원은 잡아야 할 것 같고, 제대로 쓰다보면 수백만-수천만원은 쉽게 깨질 것 같다.

## 코드에서 접속하기 (Node.js)

프로그램에서 spanner에 접속해서 사용하려면, 보통 전용 모듈을 설치하고, gcloud로 인증을 먹인 상태에서 써먹으면 된다.  
Node.js의 경우에는 다음 모듈이 필요하다.  

```javascript
npm i @google-cloud/spanner
```

아래는 Singers와 Albums  테이블을 만드는 간단한 프로그램이다.  

```javascript
// Imports the Google Cloud client library
const {Spanner} = require('@google-cloud/spanner');

/**
 * TODO(developer): Uncomment the following lines before running the sample.
 */
const projectId = '프로젝트ID';
const instanceId = 'test-spanner';
const databaseId = 'test-db';

// Creates a client
const spanner = new Spanner({
  projectId: projectId,
});

const instance = spanner.instance(instanceId);

// Note: Cloud Spanner interprets Node.js numbers as FLOAT64s, so they
// must be converted to strings before being inserted as INT64s
const request = {
  schema: [
    `CREATE TABLE Singers (
      SingerId    INT64 NOT NULL,
      FirstName   STRING(1024),
      LastName    STRING(1024),
      SingerInfo  BYTES(MAX),
      FullName    STRING(2048) AS (ARRAY_TO_STRING([FirstName, LastName], " ")) STORED,
    ) PRIMARY KEY (SingerId)`,
    `CREATE TABLE Albums (
      SingerId    INT64 NOT NULL,
      AlbumId     INT64 NOT NULL,
      AlbumTitle  STRING(MAX)
    ) PRIMARY KEY (SingerId, AlbumId),
    INTERLEAVE IN PARENT Singers ON DELETE CASCADE`,
  ],
};

async function main() {
    // Creates a database
    const [database, operation] = await instance.createDatabase(
        databaseId,
        request
    );

    console.log(`Waiting for operation on ${database.id} to complete...`);
    await operation.promise();

    console.log(`Created database ${databaseId} on instance ${instanceId}.`);
}

main()
```

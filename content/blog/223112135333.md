![](/images/img_920baf1dd7fc5cf5.png)
Scylladb는 초대형 분산 시스템을 지원하는 K/V 기반의 NoSQL 데이터베이스 중 하나다.  

2015년에 나왔고, 카산드라DB와 대부분의 기능이 호환된다.  
더 높은 성능으로 카산드라를 대체하는게 목적이고, 잘 달성하고 있는 것 같다.  

Go와 Java 천국인 요즘 DB 세상을 거슬러서 C++로 만들어졌는데, 그 덕분에 메모리 관리나 성능 측면에서 매우 안정적이라는 평가를 받는다.  
카산드라와 비교하면 동일 스펙에서의 처리량이 몇배에서 몇십배까지도 차이가 난다고 한다.  

디스코드가 중반까지는 카산드라DB로 데이터를 쌓다가 데이터가 조 단위를 넘어가자 Java 특유의 한계를 느껴서 Scylladb로 옮긴 상태다.  
이걸로 수십조단위의 메세지를 관리하고 있다고 한다.

## Docker로 시작하기

다음과 같이 실행하면, 바로 scylla 노드를 띄울 수 있다.  

```javascript
docker run --hostname some-scylla -p 9042:9042 scylladb/scylla
```

## Client 작성하기 (Go)

한번 클라이언트 프로그램을 작성해서 잘 돌아가나 테스트를 해보자.  

카산드라 호환이라서 라이브러리들도 카산드라 버전을 가져다가 쓰면 된다.  

아래는 키스페이스와 테이블을 생성하고, 데이터를 삽입한 뒤에 바로 select로 꺼내오는 간단한 코드다.  

```javascript
package main

import (
	"fmt"
	"log"
	"time"

	"github.com/gocql/gocql"
)

func main() {
	cluster := gocql.NewCluster("127.0.0.1:9042")

	session, err := cluster.CreateSession()

	if err != nil {
		log.Fatal(err)
	}

    // 키스페이스 생성
	if err = session.Query("CREATE KEYSPACE IF NOT EXISTS test_space WITH REPLICATION = { 'class' : 'SimpleStrategy', 'replication_factor' : '1' };").Exec(); err != nil {
		log.Fatal(err)
	}

    // 테이블 생성 
	if err = session.Query(`CREATE TABLE IF NOT EXISTS test_space.item(
		uuid text PRIMARY KEY,
		item_name text,
		item_price double,
		last_update_timestamp timestamp
	);`).Exec(); err != nil {
		log.Fatal(err)
	}

    // insert
	if err = session.Query(`
		insert into test_space.item(uuid, item_name, item_price, last_update_timestamp)
		values
		('c714d095-0a63-4c1f-921d-42abd954e2c1', 'buldak', 100.5, toTimeStamp(now()));
	`).Exec(); err != nil {
		log.Fatal(err)
	}

    // insert
	if err = session.Query(`
		insert into test_space.item(uuid, item_name, item_price, last_update_timestamp)
		values
		('02773666-f483-4749-a3dd-c526e62daca2', 'eojuk', 4000, toTimeStamp(now()));
	`).Exec(); err != nil {
		log.Fatal(err)
	}

	var uuid string
	var item_name string
	var item_price float64
	var last_update_timestamp time.Time

    // select 
	iter := session.Query(`select * from test_space.item;`).Iter()

	for iter.Scan(&uuid, &item_name, &item_price, &last_update_timestamp) {
		fmt.Println("Row:", uuid, item_name, item_price, last_update_timestamp)
	}

	if err := iter.Close(); err != nil {
		log.Fatal(err)
	}
}
```
카산드라와 똑같이 키스페이스->테이블의 레이어를 가지고, SQL 스타일의 구문을 지원하는 것을 볼 수 있다.

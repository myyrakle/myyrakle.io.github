Github 같은 환경에서 오픈소스 활동을 하다보면, 다른 원격 레포지토리로 포크따서 PR을 날릴 경우가 많다.  
그럴때 그걸로 전환하는 방법을 정리해보겠다.  

![](/images/img_ddb2e1875506af61.png)
이런 식으로 날라온다.  

먼저 remote를 추가하고, 그걸 통째로 fetch를 해야 한다.  

```javascript
git remote add 리모트명 https://github.com/...경로
git fetch 리모트명
```

![](/images/img_4431dd67edba3cd2.png)
그러고 나서  

리모트/브랜치의 형태로 checkout을 하면 전환이 된다.  

```javascript
git checkout 리모트명/브랜치
```

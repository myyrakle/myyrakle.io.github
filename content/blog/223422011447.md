## 설치 (Ubuntu)

우분투의 경우에는 보통 기본 레포지토리에 포함이 되어있어서 바로 설치할 수 있다.  

```javascript
sudo apt install gfortran -y
```

![](/images/img_3bccc1359da36efe.png)

깔아서 gfortran이라는 명령이 잘 실행되면, 문제없이 설치된 것이다.  

![](/images/img_cda087d28858e223.png)

## Hello World 프로그램

먼저 다음과 같이 소스파일을 하나 생성해서 작성한다.  

![](/images/img_bd2154fd2608bf72.png)

```javascript
program main
    print *, "Hello, World!"
end program main
```
오래된 노인정 언어답게 파일포맷도 통일되어있지 않다.  
보통 f90, for, f 등을 쓴다.  

program은 하나의 프로그램 로직을 뜻한다. 소스코드당 하나씩만 존재해야하는 것 같다.  
프로그램 영역 내에서는 실질적으로 실행될 코드들을 포함할 수 있다.  

코드를 잘 만들었다면, 컴파일러로 컴파일해서 바로 실행해볼 수 있다.  

![](/images/img_13a524d81ec5cabf.png)
옵션을 복잡하게 줄 것은 없다. 그냥 소스코드를 입력 파일로 넣어주기만 하면 된다.

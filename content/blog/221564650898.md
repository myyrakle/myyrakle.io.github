기본 데이터 제어  

삽입  
insert into 테이블명(속성명들...) values(값들...)  

수정  
update 테이블명 set 속성명=값, ... where 조건문  

삭제  
delete from 테이블명 where 조건문  

검색  
select 속성명... from 테이블명 where 조건문  

<br>

테이블 생성/삭제  
create table 테이블명(...)  
drop table 테이블명  

데이터베이스 생성/삭제  
create database 디비명  
drop database 디비명  

<br>

테이블에 속성 추가/삭제  
alter table 테이블명 add 속성내용...  
alter table 테이블명 drop column 속성명  

<br>

<br>

복합 조건문(if-else)  
case //시작  
when 조건식 then 반환식  
when 조건식 then 반환식  
...  
else 반환식  
end //끝

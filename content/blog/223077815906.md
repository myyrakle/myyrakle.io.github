R2는 Cloudflare의 파일 스토리지 서비스다.  
주로 이미지같은걸 올려쓸때 사용한다.  

S3와 완전 호환된다고 한다.  

비용도 매우 저렴하고, 성능도 나쁘지 않아서 static file serving에는 이만한게 없는 것 같다.  
개인적으로 S3에 올려쓰던것도 점차 옮길까 생각중이다.

<br>

<br>

## 비용

AWS S3 등의 타사 스토리지에 비하면 압도적으로 저렴한 편이다.  

<br>

![](/images/img_5f036bf3a2c0ab74.png)
일단, 리소스를 조회할 때 발생하는 데이터 전송은 무료다. 다른 서비스들은 대부분 유료다.  

스토리지는 10기가까지는 무료고, 그 이상부터 기가당 0.015달러=20원이다. AWS S3의 경우는 무료 사용량 없이 기가당 0.025 달러로 더 비싸다.  

Class A Operation은 파일 업로드 같은 가변작업 요청이다. 100만건당 4.5달러=6000원  
Class B Operation은 읽기작업이다. 100만건당 0.36 달러=500원  

자세한건 상세 사이트를 참고한다.  
[Pricing · Cloudflare R2 docs](https://developers.cloudflare.com/r2/pricing/)

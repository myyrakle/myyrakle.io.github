HAProxy를 쓰면 어렵지 않게 프록시를 구성할 수 있다.  

HTTP/HTTPS와 관계없이 단순하게 그대로 넘기는 프록시를 구성하고 싶다면, 다음과 같이 대강 세팅하면 된다.  

```javascript
frontend opensearch-front
    mode tcp
    bind *:9200
    default_backend opensearch-back

backend opensearch-back
    balance leastconn
    mode tcp

    server server0 엔드포인트:443 check
```

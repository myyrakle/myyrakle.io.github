nginx에는 여러 용도가 있지만, 대표적인 기능 중 하나가 파일들을 웹서버로 서빙하는 것이다.  

이걸 활용한다면 간단한 로컬 이미지 서버를 띄울 수 있다.  

![](/images/img_2768bc16e17bb0fe.png)

![](/images/img_1ce651af242bdb3b.png)
이미지 저장경로를 적당히 마련해두고  

nginx를 설치, 설정파일 경로를 찾아서  

<br>

```javascript
sudo apt-get install nginx
```

![](/images/img_ff003054cff0599d.png)

```javascript
sudo vi /etc/nginx/nginx.conf
```

적당히 이런식으로 편집한다.  

<br>

```javascript
events {
    worker_connections 1024;
}

http {
    include mime.types;
    default_type application/octet-stream;

    server {
            listen 80;

            location /images/ {
                alias /home/user/data0/images/;
                try_files $uri $uri/ =404;
            }
    }
}
```

![](/images/img_bef87fa8f6ebe2f3.png)

그리고 재부팅하면 된다.  

```javascript
sudo systemctl restart nginx
```

![](/images/img_e3723bf9f3d72870.png)

그래서 조회되면 잘 된 것이다.

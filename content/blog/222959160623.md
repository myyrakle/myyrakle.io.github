[https://www.npmjs.com/package/n](https://www.npmjs.com/package/n)  

그냥 지웠다 다시 까는 것도 방법일 수 있지만, 그냥 npm 도구를 통해서도 꽤나 간편하게 버전을 조절할 수 있다.  
먼저 버전 제어 도구인 n이란 녀석을 설치한다.  

```javascript
npm i -g n
```

최신버전을 깔고 싶다면 latest를 넣어주면 된다.  

```javascript
n latest
```

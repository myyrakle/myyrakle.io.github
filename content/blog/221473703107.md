제대로 쓰는 방법은 커맨드에서 gcc로 컴파일할때처럼 명령인자를 보내는 것이다.

![](/images/img_211dff79359c3d62.jpg)
node 소스파일...  
형식으로 호출하면 된다.

![](/images/img_ab8fd0926a8a2b99.jpg)

근데 계속 이렇게 메모장으로 작업을 할순 없지 않은가?  
제대로 작업을 하려면 에디터를 써야하는데 나는 Visual Studio Code를 사용하므로 이를 기준으로 기술하겠다.  
Visual Studio에도 node.js 확장이 있는 것 같던데 써보질 않아서 어떤지는 모르겠다.  

일단 문법 체크를 위하여 node.js 확장을 설치해줘야 한다.  
몇개 더 있는데 난 저걸로 골랐다.  

![](/images/img_23d07785ebe8c51c.jpg)
작업 폴더를 잘 만들고  
launch.json을 열어서 아래와 같이 만든다.  

![](/images/img_b84bc000d450a38d.jpg)

```javascript
{
    "version": "0.2.0",
    "configurations": [
        {
            "type": "node",
            "request": "launch",
            "name": "프로그램 시작",
            "program": "${file}",
            "cwd": "${workspaceRoot}"
        },
        {
            "type": "node",
            "request": "attach",
            "name": "프로세스에 연결",
            "port": 5858
        }
    ]
}
```

잘 저장하고, 이제 컨트롤+f5를 눌러서 실행이 잘 되면  

![](/images/img_131511bc0f5be9fc.jpg)
세팅 완료다.
